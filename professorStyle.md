# 병태생리학 시험 문항 유형 분석 템플릿

---

## 1) 기본형

### A. 문항 구조 템플릿

- **[질문문] + 5지선다**가 기본
- 자주 쓰는 질문문 패턴:
  - ~에 대한 설명이다. **옳지 않은 것은?** (부정형/오답 고르기)
  - ~에 해당하는 것은? (정의 매칭)
  - ~의 특징이다. **옳지 않은 것은?** (특징 리스트 + 예외)
  - ~이다. **옳은 것을 모두 고르면?** (복수선택)
- 지문 길이: **짧음~중간** (한 문단 + 선택지)

### B. 난이도/인지 수준

- **기억/이해**(정의·특징) 비중 높음
- 기전 기반(원인→결과 / 호르몬→장기기능 / 압력차) 일부 포함
- 유전/면역/종양/쇼크처럼 **범주를 알고 분류하는 문제** 많음

### C. 선택지(보기) 스타일

- 보기들이 **문장형 설명(서술문)**인 경우 많음
- 함정 포인트:
  - "정상/비정상" 판단 뒤집기
  - 수치/방향(**증가·감소**)로 오답 유도
  - 용어가 비슷한 것(급성/만성, 거부반응 유형 등) 섞기
  - 영문 병기 종종 있음 (예: shock 유형)

### D. 출제자 톤 (문장/표현 습관)

- 교과서 문장 그대로 가져온 듯한 **건조한 서술체**
- `( )` 빈칸 넣어서 장기/개념 맞추기도 사용
- 임상 맥락은 있어도 **"정의/설명 제시형"**에 가깝고, 케이스 디테일은 적음

### E. 타입 분류 (1~40에서 섞이는 타입들)

| 타입 | 설명 |
|------|------|
| **NEGATIVE** | 옳지 않은 것 (예외 찾기) |
| **DEFINITION-MATCH** | 설명 → 개념/질환 매칭 |
| **MECHANISM** | 기전/호르몬/압력경사 → 결과 |
| **MULTI-SELECT** | "모두 고르기" |
| **CLASSIFICATION** | 쇼크/염증/종양/유전/면역 유형 분류 |

## 2) 임상 케이스형

### A. 문항 구조 템플릿

- 케이스 한 덩어리:
  > **연령/성별** + **사건/질환** + **시간경과** + **증상/검사**
- 마지막 한 문장:
  - *이때 가장 적절한 것은?*
  - *주로 관찰되는 것은?*
  - *주요 매개물질은?*
- 보기 **4지** (또는 짧은 리스트)로 단답형에 가까움

### B. 핵심 특징

- **증상·검사치**(WBC, 발열 등)가 등장 → 상황판단형
- 답이 **대표 키워드 1개**로 떨어짐:
  - 급성 염증 → **호중구**
  - 즉시형 반응 → **히스타민**
  - 결핵 → **육아종**
- 문장 톤이 더 **현장형(케이스 리딩)**이고, 기본형 같은 "~에 대한 설명이다" 교과서 톤이 덜함

### C. 난이도/인지 수준

- **대표 연결(클리셰 매칭)** 중심: *"이 상황이면 이거"*
- 계산/복잡한 추론보단 **pattern recognition**
