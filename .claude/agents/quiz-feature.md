# Quiz Feature Agent

퀴즈 관련 기능을 담당하는 에이전트입니다.

## 역할

- 퀴즈 목록 화면 (카드 2열, 필터 탭)
- 퀴즈 풀이 화면 (OX, 객관식, 주관식)
- 결과 화면 (줌아웃, 점수, 골드 획득)
- 피드백 입력 화면
- 자체제작 퀴즈 (Tesseract.js OCR)
- 퀴즈 PDF 다운로드 (jsPDF)
- TOP3 레이스 애니메이션

## 퀴즈 유형

### OX 문제
```
│     ┌─────────┐   ┌─────────┐     │
│     │    ⭕   │   │    ❌   │     │
│     └─────────┘   └─────────┘     │
```

### 객관식 (4지선다)
```
│  ┌────────────────────────────┐   │
│  │ ① 선지 1                   │   │
│  └────────────────────────────┘   │
│  ┌────────────────────────────┐   │
│  │ ② 선지 2                   │   │
│  └────────────────────────────┘   │
│  ... (3, 4번)
```

### 주관식
```
│  ┌────────────────────────────┐   │
│  │ 답 입력...                 │   │
│  └────────────────────────────┘   │
```

## 화면 구성

### 퀴즈 목록
- 필터 탭: [전체] [중간] [기말] [족보] [자체제작]
- 시즌 날짜 따라 기본 탭 자동 설정
- TOP3 레이스 (토끼가 말 타고 달리기)
- 반 참여도 순위 바

### 퀴즈 풀이
- 상단: 뒤로가기 + 제목 + 진행도 (3/10)
- 중단: 문제 + 선지 + 즉시 피드백 버튼 (❗)
- 하단: [이전] [다음] 버튼

### 결과 화면 (줌아웃)
- 제목 + 획득 골드
- 소요 시간 + 점수
- 문제별 정답/오답 표시
- 오답은 자동으로 복습창 저장
- 📚 버튼으로 찜하기

### 피드백 화면
- 전체 문제 + 선지 + 내 답 + 해설 스크롤
- 각 문제마다 피드백 입력칸
- [Skip] [완료 💰+15] 버튼

## 자체제작 퀴즈

- Tesseract.js로 사진/PDF → 텍스트 추출
- 추출된 텍스트 양식 자동 입력
- 사용자 검수 후 저장
- 다른 학생이 풀면 제작자에게 골드 +2

## 애니메이션

- Framer Motion: 페이지 전환, 선지 선택
- Lottie: TOP3 레이스, 결과 축하

## 주의사항

- 오프라인 모드에서도 퀴즈 풀기 가능 (Service Worker)
- 온라인 복귀 시 자동 동기화
- 정답 노출 방지 (서버에서 채점)
