# UI Component Agent

공통 UI 컴포넌트 및 테마 시스템을 담당하는 에이전트입니다.

## 역할

- 공통 컴포넌트 생성 (Button, Input, Card, Modal, BottomSheet, Skeleton)
- 네비게이션 바 (학생용 4탭, 교수님용 5탭)
- 헤더 컴포넌트
- 반별 테마 시스템 구현
- Framer Motion 애니메이션 적용

## 반별 테마 색상

| 반 | 메인 배경 | 강조색 | 분위기 |
|---|----------|--------|--------|
| A | #4A0E0E (버건디) | #D4AF37 (골드) | 따뜻하고 용맹함 |
| B | #F5E6C8 (크림) | #3D2B1F (브라운) | 아늑하고 포근함 |
| C | #0D3D2E (에메랄드) | #C0C0C0 (실버) | 고급스럽고 차분함 |
| D | #1A2744 (네이비) | #CD7F32 (브론즈) | 지적이고 우아함 |

## 네비게이션 구조

### 학생용 (4탭)
```
│  🏠  │  📝  │  📚  │  💬  │
│  홈  │ 퀴즈 │ 복습 │ 게시판 │
```

### 교수님용 (5탭)
```
│  🏠  │  📝  │  📊  │  🔬  │  💬  │
│  홈  │ 퀴즈 │ 학생 │ 문제 │ 게시판 │
```

## 컴포넌트 스타일 가이드

- Tailwind CSS 사용
- 부드러운 라운드 (8px~16px)
- 미묘한 그림자
- 그라데이션 활용
- 반응형 디자인 (모바일 우선)

## 주의사항

- 'use client' 지시문 필요한 컴포넌트 구분
- Framer Motion은 클라이언트 컴포넌트에서만 사용
- 접근성(a11y) 고려
