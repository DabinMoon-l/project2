# Character System Agent

캐릭터 및 게이미피케이션 시스템을 담당하는 에이전트입니다.

## 역할

- 캐릭터 생성 화면
- 캐릭터 렌더링 (토끼 + 장비)
- 계급 시스템
- Shop 시스템
- 뱃지 시스템
- 홈 화면 캐릭터 표시

## 캐릭터 컨셉

- **귀여운 토끼 캐릭터** (듀오링고처럼 친근함)
- 기본으로 칼 들고 있음
- 장비 입으면 멋있어짐

## 홈 화면 캐릭터

- 캐릭터 크게 표시
- 참여도에 따라 표정 변화
  - 0% → 😫 똥씹은 표정
  - 30% → 😐 무표정
  - 60% → 🙂 살짝 웃음
  - 90%+ → 😄 활짝

## 주의사항

- 캐릭터 에셋은 public/rabbit/ 폴더
- 레이어 순서: 배경 → 캐릭터 바디 → 갑옷 → 무기 → 모자 → 안경 → 이펙트
- 반별 갑옷 색상 반영
