# 🧬 생물학 퀴즈 생성 가이드

> **목적**: AI 퀴즈 생성 시 각 챕터별로 집중해야 할 핵심 포인트를 정의합니다.
> **대상**: 간호학과 1학년 생물학 과정

---

## 2장. 세포의 특성

### 퀴즈 포커스

- **비교 문제 출제**: 주사전자현미경(SEM) vs **투과**전자현미경(TEM) — 각각의 원리와 관찰 대상 차이
- **비교 문제 출제**: 원핵세포 vs 진핵세포 — 구조적 차이점 (핵막, 세포소기관 유무 등)
- **기능 매칭**: 세포소기관별 역할 (미토콘드리아, 리보솜, 골지체, 소포체 등)
- **구조-기능 연결**: 세포막의 구조(인지질 이중층, 막단백질)와 물질이동 방식
- **비교 문제 출제**: 수동수송 vs 능동수송 — ATP 필요 여부, 농도 기울기 방향
- **간단 개념 확인**: 세포골격의 기본 구성과 역할

## 3장. 생명체의 화학적 이해

### 퀴즈 포커스

- **물의 특성**: 분자 형태(극성), 응집력, 높은 비열 등 생물학적 중요성
- **결합 유형 구분**: 공유결합 vs 이온결합 vs 수소결합 — 강도와 특성 비교
- **삼투압 적용**: 저장액 vs 고장액에서 세포의 변화 (용혈, 수축 등)
- **당질 분류**: 단당류 → 이당류 → 다당류 계층 구조
- **단백질 핵심**:
  - 입체구조 중 **삼차구조** 강조 — 단백질 접힘에 관여하는 결합(수소결합, 공유결합 등)
  - **단백질 변성** — 원인과 결과
- **핵산 비교 (고빈도 출제)**:
  - DNA vs RNA 비교표: 당(deoxyribose vs ribose), 염기(티민 vs 우라실), 구조(이중나선 vs 단일가닥), 존재 위치(핵 vs 세포질+핵)
  - ATP 구조 — 인산기 개수에 따른 ATP/ADP/AMP 구분
  - DNA 구조 — 사슬 방향성(5' → 3'), 상보적 염기쌍(A-T, C-G)
  - 뉴클레오타이드 구조 — 인산기 + 당 + 염기

## 4장. 영양과 대사

### 퀴즈 포커스

- **소화 과정 순서 및 효소 매칭**:
  - 단백질: 펩시노겐 → 펩신 (pH 2, 위에서 활성화)
  - 당질: 타액 아밀레이스 → 이자 아밀레이스 → 말테이스 (순서 중요)
  - 지질: 쓸개즙산(간 생성, 쓸개 저장, 지질 유화) → 이자 라이페이스(중성지방 → 지방산 + 모노글리세리드) → 미셀 → 킬로미크론
- **킬로미크론 흡수 경로**: 혈액 직접 흡수 ✕ → **림프관** 흡수 후 혈액으로 이동
- **개념 구분**: 이화작용 vs 동화작용
- **ATP 구조**: 인산~인산 간 **고에너지 결합** 위치
- **효소 특성**:
  - 단백질로 구성 → 온도, 염, pH 변화에 따른 활성 변화 (그래프 해석 가능)
- **효소 조절 메커니즘**:
  - 음성피드백(항상성 유지) vs 양성피드백
  - 알로스테릭 조절
  - 한정분해: 프로트롬빈 → 트롬빈
  - 부가에 의한 조절: 키나이제(인산기 부착 → 활성화)
- **에너지대사 경로와 장소 매칭 (고빈도)**:
  - 해당경로 → 세포질
  - TCA 회로 → 미토콘드리아 기질
  - 산화적 인산화(전자전달계 + ATP 합성) → 미토콘드리아 내막
  - 전자전달계 최종 전자 수용체 = **산소**
- **혐기적 ATP 합성**: 피루브산 → 젖산 (목적: NAD⁺ 재생), 발효

## 5장. 유전과 DNA

### 퀴즈 포커스

- **용어 구분**: 상동염색체 vs 염색분체
- **멘델의 법칙 (간단 개념 확인)**:
  - 우열의 원리, 분리의 법칙
  - 독립의 법칙 — 조건: 서로 다른 염색체에 존재, 연관 ✕
- **교차**: 감수분열 동안 발생
- **연쇄와 재조합**: 기본 개념
- **DNA 복제 핵심**:
  - 관여 단백질: helicase, DNA polymerase, DNA ligase 등 역할 매칭
  - 복제 방향: 5' → 3'
- **간단 개념 확인**: PCR, 텔로미어
- **염기 개수 추론 문제 (계산형 출제)**:
  - 예시: 이중나선 DNA 300개 염기 중 G가 50개일 때 A의 개수 구하기
  - 핵심 원리: A=T, G=C (상보적 염기쌍 규칙)
- **염색체 관련 용어 구분 (고빈도)**:
  - 염색질 → 염색사 → 염색체 → 염색분체 관계
  - 동원체, 뉴클레오좀, 히스톤 단백질의 역할

## 6장. 유전자의 발현

### 퀴즈 포커스

- **센트럴 도그마 흐름**: DNA → RNA(전사, 핵에서) → 단백질(번역, 세포질에서)
- **전사 과정**:
  - RNA 중합효소의 역할 (DNA 변성, 뉴클레오티드 연결)
  - 전사 방향: 5' → 3'
  - **주형가닥 vs 비주형가닥**: RNA는 비주형가닥의 염기서열이 베껴짐
- **RNA 종류와 역할**:
  - **mRNA** (유전정보 전달), rRNA (리보솜 구성), **tRNA** (아미노산 운반)
  - 추가 역할: 리보자임, 스플라이싱
- **번역 과정**: 전체 흐름 이해
- **유전암호 (고빈도)**:
  - 시작코돈: AUG → **메티오닌**
  - 종결코돈: UAA, UAG, UGA
- **실전 문제 유형 (반드시 출제)**:
  - 주어진 염기서열에서 → (1) 코돈 식별 (2) 해당 아미노산 (3) 안티코돈 도출

## 7장. 세포의 주기와 죽음

### 퀴즈 포커스

- **세포주기 단계별 특징**: G₁기, S기(**DNA 복제**), G₂기, M기(**핵분열 + 세포질분열**)
- **핵분열 단계별 특징**: 전기 → 중기 → 후기 → 말기 (각 단계의 핵심 사건)
- **감수분열 비교 (고빈도)**:
  - 감수분열 I vs 감수분열 II — 핵상 변화, 염색체 수 변화
  - 전기 I: **2가 염색체 형성**, **교차 발생**
  - 후기 I: **상동염색체의 분리**
  - 후기 II: **염색분체의 분리** (= 체세포 분열과 동일)

## 8장. 생식·발생·분화

### 퀴즈 포커스

- **비교 문제 출제**: **유성생식** vs 무성생식
- **초기 발생 순서**: 수정란 → 상실배 → 포배(포배강 존재)
- **낭배 이후 기관형성 — 배엽별 분화 (고빈도)**:
  - 외배엽: 표피, **신경**
  - 내배엽: 내장 (세포들이 낭배 안으로 이동하여 형성)
  - 중배엽: 결합조직, 근육
- **사람의 생식계**: 남성(음경, 전립샘, 정관, 고환) / 여성(난관, 난소, 자궁, 자궁목, 질) 구조
- **정자 구조와 기능 매칭**:
  - 첨체: 가수분해효소 → 난소 보호막 관통
  - 편모: 운동
  - 미토콘드리아: ATP 생산 → 운동 에너지
- **난자**: 극체의 특징 (기능 없음)
- **수정 (고빈도)**:
  - 수정 장소: **팽대부**
  - 과정: 1차 난모세포 → 2차 난모세포 + 정자 수정 → 성숙난자
- **태반과 태아**: 태반의 구조와 혈액 흐름 이해
- **줄기세포**: 자기복제 + 분화세포 생성

## 9장. 동물의 조직

### 퀴즈 포커스

- **상피조직**: 내분비샘(호르몬 생성) vs 외분비샘 구분
- **지지조직**:
  - 피부의 조직 구조 구성 요소
  - 결합조직 특성
  - 뼈조직: Ca 풍부, 조골세포 vs 파골세포 (항상성 유지)
  - 연골조직: 혈관 없음 → 확산으로 영양공급, 콜라겐 풍부
- **근육조직 분류표 (고빈도)**:
  - 뼈대근육 / 심장근육 / 민무늬근육
  - 맘대로근(수의근) vs 제대로근(불수의근) 구분
- **뼈대근육의 구조**:
  - 근원섬유, I대(**수축 시 길이 감소**), A대(**길이 변화 ✕**), H대
  - 미오필라멘트, 액틴필라멘트
- **근육 수축 기전 (순서 중요)**:
  - Ca²⁺ 방출 → Ca²⁺-트로포닌 결합 → 구조 변화 → 트로포미오신 이동 → 액틴-미오신 상호작용
- **근육 에너지대사**: 혐기성 상태 → 젖산 생성, 코리회로 (간단히)
- **혈액 구성 (고빈도)**:
  - **55% 혈장 + 45% 혈구**
  - 혈장 vs 혈청 차이
- **혈액 세포 성분 비교표**:
  - 적혈구 / 혈소판 / 백혈구 — 핵 유무, 생성장소, 파괴장소, 주요 기능
- **혈액 액체 성분**: 혈장 구성요소(단백질 등)와 기능
- **혈액형 (고빈도)**:
  - ABO식: 항체-항원 관계
  - **Rh 부적합**: Rh⁻ 어머니 + Rh⁺ 태아 → 응집반응 발생 기전
- **간단 개념 확인**: 혈액세포 분화, 혈액 응고계(응고 + 섬유소용해)

## 10장. 동물의 기관

### 퀴즈 포커스

- **소화계**: 각 기관의 **위치** 파악, 내장근의 꿈틀운동
- **문맥 개념**: 심장과 무관, 기관과 기관을 연결 — **간문맥** 강조
- **소장**: 융모 多 → 표면적 확대 → 영양 흡수 효율 향상
- **영양소 흡수 경로 비교 (고빈도)**:
  - 지용성: 융모세포 → 유미관(암죽관) → 림프관 → 정맥 → 심장 → 온몸
  - 수용성: 융모세포 → 소장 모세혈관 → **간문맥** → 간 → 간정맥 → 하대정맥 → 심장 → 온몸
- **간의 기능**: 쓸개즙 생성, 요소회로, 합성 기능 등
- **이자(췌장)**: 소화기관 + 외분비/내분비 기관 이중 역할
- **호흡계 구조**: 인두 → 후두 → 기관 → 주기관지 순서
- **가스교환 (고빈도)**:
  - 허파: 고산소 상태 → O₂가 헤모글로빈과 결합 → 조직으로 이동
  - 조직: 저산소 상태 → CO₂ 대부분 **중탄산이온**으로 혈장 통해 허파로 이동 후 배출
- **심장 구조**: 심방, 심실, 반달판막, 이첨판, 삼첨판
- **혈관 구조**: 아래대정맥, 위대정맥, 허파동맥, 허파정맥
- **자극전달계 (순서 중요)**: 동방결절 → 방실결절 → 히스다발 → 푸르킨예섬유 → 수축
- **순환 비교**: 폐순환(O₂ 흡수, CO₂ 배출) vs 체순환(O₂ 공급, CO₂ 회수)
- **혈관 비교**: 동맥(두꺼움, 탄력) vs 정맥(판막, 근육)
- **혈압 순서**: 대동맥 > 세동맥 > 모세혈관 ≥ 정맥
- **림프계**: 지질 운반, 림프절(면역반응), 빗장밑정맥으로 합류
- **콩팥 — 네프론 기능 (고빈도)**:
  - 소변 생성 과정:
    1. 토리 여과: 물, 포도당, 아미노산 등 여과 (단백질, 지방, 적혈구는 여과 ✕)
    2. 콩팥요세관 재흡수: 포도당, 아미노산, 물, 요소 등
    3. 콩팥요세관 분비: 암모니아, 크레아티닌
  - 원뇨 → 소변 변화:
    - 사라지는 것: 단백질, 글루코스(100% 재흡수)
    - 농축되는 것: 요소, 요산, 크레아티닌, 암모니아, 무기염류

## 11장. 내분비계

### 퀴즈 포커스

- **항상성 유지 기전**: 음성 피드백 vs 양성 피드백
- **신경계 vs 내분비계 비교**: 빠름/일대일 vs 느림/다중 반응
- **호르몬 3종류 구분**: 펩티드호르몬, 스테로이드호르몬(**지용성**), 아민/아미노산호르몬
- **호르몬의 일반적 특징**:
  - 내분비샘에서 분비
  - 표적 기관에만 작용
  - 미량으로 조절 가능
  - 척추동물 간 종 특이성 없음
- **호르몬 분비 기관 매칭 (고빈도)**:
  - 시상하부, **뇌하수체**, **갑상샘**, **부갑상샘**, 심장, 위, 샘창자, **이자**, 난소, 고환, 간, **콩팥위샘**, **콩팥**
- **호르몬 작용 기전**: 간단히
- **기관별 호르몬 분류 (고빈도)**:
  - 시상하부 호르몬
  - 뇌하수체 전엽 / 후엽 호르몬
  - 갑상샘 / 부갑상샘 호르몬
  - 이자 호르몬 (인슐린, 글루카곤)
  - 부신 겉질 / 속질 호르몬
- **항상성 유지 시나리오 문제**:
  - 염분과 수분 조절
  - 혈당량 조절
  - 체온 조절
  - 칼슘이온 조절

## 12장. 신경계

### 퀴즈 포커스

- **신경계 유래**: 외배엽
- **신경계 구성 세포**: 뉴런 + 신경아교세포
- **신호 전달 방식 구분**:
  - 세포체 → 축삭 말단: **전기적 신호**
  - 축삭 말단 → 수용체(시냅스): **화학적 신호**(신경전달물질)
- **도약 전도 (고빈도)**: 랑비에결절에서 도약, 말이집(수초)이 절연 역할
- **전도 vs 전달 구분**:
  - 흥분전도: 뉴런 내, 전기적 신호
  - 신경전달: 뉴런 간 시냅스, 화학적 신호
- **신경계 분류 체계**:
  - 중추신경계: 뇌(뇌신경 12쌍), 척수(척수신경 31쌍)
  - 말초신경계: 들신경(감각기관 → 중추), 날신경(중추 → 근육/샘세포)
  - 운동신경: 몸운동신경 + 자율신경
- **뇌 구성과 역할 매칭 (고빈도)**:
  - 대뇌: 수의운동, 고등사고 / 겉질(세포체, 가지돌기) vs 속질(축삭돌기)
  - 소뇌: 근육운동 조화
  - 사이뇌: 시상, 시상하부(자율신경 중추 — 내장, 체온, 혈당량, **항상성**, 뇌하수체 호르몬 분비)
  - 중간뇌: 안구운동
  - 다리뇌 및 숨뇌: 호흡, 순환, 신경교차(얼굴 신경은 교차 ✕)
- **사이신경세포**: 척수반사 — 감각신경에 대해 운동신경이 직접 반응
- **자율신경계 비교표 (고빈도)**:
  - 교감신경: 동공확대, 박동촉진, 혈관수축/혈압상승, 배뇨억제, 소화관억제
  - 부교감신경: 동공축소, 박동억제, 혈관이완/혈압하강, 배뇨촉진, 소화관자극
- **자율신경 세부 비교**:
  - 신경절 이전 뉴런 길이: 교감(짧) vs 부교감(긺)
  - 신경전달물질: 교감(**노르에피네프린**) vs 부교감(**아세틸콜린**)
- **척수 신경 연결**: 감각신경(등쪽), 사이신경세포, 운동신경(배쪽)
- **활동전위 단계 (고빈도)**:
  - 분극(-60mV): Na⁺-K⁺ 펌프(능동수송, Na⁺ 밖, K⁺ 안) + K⁺ 통로(촉진확산, K⁺ 일부 유출)
  - 탈분극(+50mV): 전위의존성 Na⁺ 통로 개방 (촉진확산)
  - 재분극: 전위의존성 K⁺ 통로 개방 (촉진확산)
  - 각 단계별 전하(음/양)와 우세 이온 구분
- **화학시냅스 전달 기전**: 간단히
- **신경전달물질 매칭 (고빈도)**:
  - 아세틸콜린: 중추, 운동뉴런, 부교감신경
  - 노르아드레날린: 중추, 교감신경
  - 세로토닌, 도파민: 역할 구분

1. **O/X 문제**: 핵심 개념의 정확한 이해 확인

6. **계산형**: 염기 개수 추론 등
7. **시나리오형**: 항상성 유지 상황에서의 호르몬 반응 등
