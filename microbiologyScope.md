# 핵심 미생물학 정리

---

## Chapter 01. 미생물과 미생물학

### 학습목표
1. 원생생물계에 포함되는 미생물의 종류를 열거할 수 있다.
2. 진핵생물과 원핵생물의 차이를 설명할 수 있다.
3. 병원성의 개념을 설명할 수 있다.
4. 미생물학의 변천과정에서 역사적 사건을 열거할 수 있다.

---

### 1. 미생물의 개요

#### 1) 미생물(microorganism)의 정의
- 육안으로는 볼 수 없는 미세한 생물
- 세균(bacteria), 바이러스(virus), 진균/곰팡이(fungi), 원충/원생동물(protozoa), 조류(algae) 등이 포함
- 크기 범위: 원생동물·조류 20~90μm ~ 바이러스 20~300nm

#### 2) 미생물의 생물학적 위치

**분류 변천사**
- **2계설**: 동물계(animal kingdom) + 식물계(plant kingdom) → 미생물을 작은 식물이나 동물로 간주
- **3계설**: 현미경 발명 후 원생생물계(protista kingdom) 독립 — 원생동물(protozoa)과 세균류(bacteriomycota) 포함
- **4계설**: 원생생물계에서 세균을 독립
- **5계설**: 4계설에서 진균을 추가 독립

**원핵생물(prokaryote) vs 진핵생물(eukaryote)**
- 세포 구조에 따라 원핵세포와 진핵세포로 나뉨
- 바이러스는 세포 형태를 갖지 않으므로 원핵생물에도 속하지 않음 → 생물과 무생물의 사이

**원핵생물과 진핵생물의 차이점**

| 구조물 | 원핵생물 (세균, 고세균) | 진핵생물 (동·식물, 진균, 원충, 조류) |
|---|---|---|
| 핵막 | − | + |
| 염색체 | 1개 (환형) | 여러 개 (1개 이상), 히스톤과 결합, 직선이며 말단에 텔로미어(telomere) |
| 핵소체 | − | + |
| 유사분열 | − | + |
| 미토콘드리아 | − | + |
| 세포질그물, 골지체 | − | + |
| 엽록체 | − | + 또는 − |
| 미세소관 (세포 내 뼈대계통) | − | + |
| 리보솜 | 70S (30S + 50S) | 80S (40S + 60S) |
| 세포벽 | + (펩티도글리칸) | 동물(−), 식물(+ 셀룰로오스), 진균(+ 키틴) |

> **참고**
> - 콜레라균이나 장염비브리오균에서는 염색체가 2개 존재하는 것으로 보고됨
> - 미토콘드리아 내의 리보솜은 70S이며, 대부분 다세포생물의 미토콘드리아 DNA는 원핵세포와 유사
> - 세포벽이 없는 특수한 것으로 마이코플라스마와 세균의 L형(L-form)이 있고, 클라미디아의 세포벽(외막)에는 펩티도글리칸이 없음

**고세균(archaeobacteria)**
- 형태적으로는 세균류와 비슷하나 유전자의 구조나 발현이 진핵생물과 유사
- 열이나 염분에 고도의 내성을 나타내거나 메탄을 생산하는 균
- 생물을 세균, 고세균, 진핵생물의 3가지 **도메인(domain)**으로 분류

**3도메인설과 5계설 대응 및 대표 생물종**

| 3도메인설 | 5계설 | 대표적 생물종 |
|---|---|---|
| 세균 | 원핵생물계 | (진정)세균 — 대장균, 결핵균 / 남조류(남조세균) — 연쇄체목, 흔들말목 |
| 고세균 | 원핵생물계 | 고도호열균, 메탄세균, 고도호염균 |
| 진핵생물 | 원생생물계 | 아메바, 연두벌레, 트리코모나스, 트리파노소마 |
| | 균계 | 털곰팡이, 효모, 푸른곰팡이, 버섯 |
| | 식물계 | 솔이끼, 고사리, 은행나무, 벚꽃나무 |
| | 동물계 | 지렁이, 오징어, 거미, 개구리, 원숭이 |

### 2. 미생물학의 역사

#### 1) 미생물학의 영역과 의의

**미생물학의 영역**
- 미생물학(microbiology): 생태계에 널리 존재하는 미생물을 연구하는 학문
- **기초미생물학(basic microbiology)**: 미생물의 형태, 구조, 증식과 대사, 유전, 자연계에서의 미생물 생태, 사람을 비롯한 다른 생물과의 관계
  - 세부: 미생물분류학, 미생물생리학, 미생물유전학, 미생물생태학, 면역학, 역학
- **응용미생물학(applied microbiology)**: 미생물의 유용한 이용을 연구
  - 세부: 환경미생물학, 해양미생물학, 토양미생물학, 식품미생물학, 공업미생물학, 감염관리학

**의학미생물학(medical microbiology)**
- 병원성(pathogenicity): 사람이나 동·식물에 질병을 일으키는 성질
- 병원성 미생물(pathogenic microorganism): 병원성을 가진 미생물
- 병원미생물학(pathogenic microbiology): 병원성 미생물을 대상으로 하는 학문
- 의학미생물학: 그중에서도 사람에게 감염을 일으키는 미생물을 대상
- 연구 대상: 각 병원성 미생물의 성질, 형태, 병원성, 전파성, 감염경로, 매개동물, 감염 예방과 제어, 감염 발병 시 병태, 진단과 치료

**미생물학의 의의**
- 인수공통감염병(zoonosis): 사람과 동물 모두에 병원성을 나타내는 미생물에 의해 발생하는 질병
- 기회감염(opportunistic infection): 건강한 사람에게는 무해한 비병원성~약병원성 미생물이 저항력 저하된 숙주에게 병원성을 나타내는 것
- 의료 종사자(의사, 간호사, 의료기사 등)는 항상 병원성 미생물과 밀접하게 접촉하며 보유자(reservoir) 또는 매개자(vector)가 될 위험성이 있음
- 신종 감염병(emerging infectious disease): 그동안 잘 알려지지 않았던 미생물에 의한 감염병
- 재출현 감염병(reemerging infectious disease): 과거에 유행하다 발생이 주춤한 뒤 다시 기세를 확장하는 감염병

#### 2) 미생물학의 변천과정

**① 감염병학의 기원**
- 중세시대까지 전염병은 '신의 벌'이라고 믿음
- 히포크라테스(Hippocrates, B.C. 459~377): 환자의 자연치유력 중시, 감염병의 원인을 '오염된 공기(miasma)'로 파악
- 14~15세기: 천연두, 페스트 대유행 → 콜럼버스의 미대륙 발견(1492) 이후 매독 유행 확산
- 프라카스토로(Fracastoro, 1478~1553): 각 전염병에는 '살아 있는 전염원(contagium vivum)'이 존재하며 직접 접촉, 매개물, 공기를 통해 전파된다고 설명 → 오늘날 감염병의 병인론(etiology)으로 이어짐

**② 현미경을 이용한 미생물의 관찰과 기록**
- 레벤후크(Leeuwenhoek, 1632~1723): 17세기 초반, 렌즈를 연마하여 광학현미경 제작
- 막대균(간균), 알균(구균), 나선균 등을 구별 → 미생물 관찰과 기록의 효시

**③ 현대 미생물학의 기초**
- **파스퇴르(Pasteur, 1822~1895)**
  - 저온살균법(pasteurization) 개발: 62~65℃에서 30분간 가열
  - 백조목 플라스크(swan neck flask) 고안 → 자연발생설의 오류를 실험적으로 증명
  - 탄저 및 광견병 백신 개발
- **리스터(Lister)**: 석탄산을 이용한 외과수술 시 소독법 고안 → 화농 방지 성공
- **레마크(Remak)**: 질병 원인 미생물을 최초 발견 — 소아 황선(favus) 환자로부터 황선균 분리
- 오베르마이어(Obermeier): 재귀열균 발견 (1868)
- 로쉬(Loesch): 이질아메바 발견 (1875)
- **코흐(Koch, 1843~1910)**
  - 탄저균, 결핵균, 콜레라균 분리
  - 순수배양법(고형배지) 확립
  - **코흐의 4가지 원칙(Koch's postulates)** 발표 (1884):
    1. 해당 질병의 병변부위에서 반드시 해당 병원체가 발견되어야 한다 (관련성)
    2. 병변부위에서 해당 병원체가 분리되고 순수배양되어야 한다 (분리성)
    3. 순수배양된 병원체를 건강한 숙주에 접종하면 동일한 질병을 일으켜야 한다 (인과성)
    4. 접종하여 질병에 걸린 숙주로부터 동일한 병원체가 분리되어야 한다 (재분리성)

**독소와 혈청학 연구**
- 루(Roux)와 예르생(Yersin): 디프테리아균 배양액에서 외독소 생산 사실 발견, 접종 동물에서 독소 중화 물질 생성 확인 (1883)
- 기타사토(Kitasato): 무산소성균 배양법 확립, 파상풍균 분리, 독소의 중요성 보고 (1889)
- 베링(Behring) + 기타사토: 디프테리아 및 파상풍의 항독소 혈청요법 논문 발표 (1890)

**바이러스의 발견**
- 이바노프스키(Iwanowski): 담배모자이크병 병원체가 세균여과기를 통과하는 미세한 것이라는 보고 (1892)
- 뢰플러(Löffler)와 프로슈(Frosch): 소의 구제역 병원체가 세균보다 작은 생물이라고 추론 (1898)
- 베이에링크(Beijerink): 담배모자이크병의 원인으로 감염성 액체(contagium vivum fluidum) 보고
- 얼(Earle): 마우스 섬유모세포(L세포) 배양 수립 (1948)
- 게이(Gey): 사람 자궁경부암세포(HeLa세포) 배양 수립 (1952)
- 엔더즈(Enders) 등: 배양세포에서 폴리오바이러스 증식 성공 (1949)

**④ 예방의학의 시작과 면역학 확립**
- **제너(Jenner, 1749~1823)**: 우두에 걸린 소와 접촉한 목동·소젖 짜는 여성이 천연두에 걸려도 증상이 가볍고 이후 면역된다는 사실 관찰 → 우두 활용 백신 개발 (1798)
- **파스퇴르**: 병원성이 약한 닭콜레라균 배양액 접종 → 가벼운 증상 후 회복, 이후 살아있는 균 접종해도 발병하지 않음 발견 (1880) → 약독화 또는 불활성화 백신 제작
- 항독소혈청(antitoxic serum) 치료 응용: 루, 예르생, 기타사토, 베링 등
- 파이퍼(Pfeiffer): 용균소 발견 (1894)
- 그루버(Gruber): 응집소 발견 (1896)
- 크라우스(Kraus): 침강소 발견 (1897)
- 뷔히너(Büchner): 보체 발견 (1895)
- 란트스타이너(Landsteiner): 항원의 면역특이성이 화학구조에 따른다는 사실 증명 (1907)
- 티셀리우스(Tiselius)와 카밧(Kabat): 항체의 본체가 γ글로불린임을 증명 (1938)
- 버넷(Burnet): 클론선택설(clonal selection theory) 발표 — 특이적 항체 생산 이론 및 자기·비자기 확인기전 (1959)

### 1. 면역계통의 개요

#### 1) 면역의 정의
- 면역(免疫, immunity): 자기(self; 숙주)와 비자기(nonself; 병원체)를 식별하여 비자기를 제거하는 생체방어시스템
- 면역계통(immune system): 생체방어시스템 자체를 가리킴

**생체방어시스템 — 3층 구조**

| 구분 | 내용 |
|---|---|
| **1차 생체방어시스템** | 가장 간단한 방어시스템. 미생물이 생체에 침입하는 것을 막는 **물리적 장벽** 등에 의함 |
| **2차 생체방어시스템 (선천면역)** | 물리적 장벽을 돌파한 병원체에 대해 특이성은 높지 않지만 다양한 병원체를 인식. **선천면역(innate immunity; 자연면역)**이라 하며, 모든 동·식물에서 확인 |
| **3차 생체방어시스템 (후천면역)** | **후천면역(acquired immunity; 획득면역)**이며, **세포면역(cellular immunity)**과 **체액면역(humoral immunity)**이 담당 |

- **세포면역**: T세포, 큰포식세포 등의 세포 무리가 중심이 되어 작용
- **체액면역**: 항체, 보체 등의 체액인자가 중심이 되어 작용
- 후천면역은 **특이적 인식**을 하는 면역계통이며, **면역기억(immunologic memory)** — 후천면역에서 작용한 면역계통이 유지되는 것 — 을 특징으로 함

> **보체(complement system)**: 척추동물의 신선한 혈청 속에서 효소와 같은 작용을 하는 물질. 병원체를 제거하기 위해 염증을 촉진하여 병원체의 세포막을 공격하는 능력을 향상시키는 면역 작용과 식작용의 기능을 보완하는 물질

#### 2) 면역반응

**초기 반응 (수분~수시간)**
- 병원체가 표피나 점막의 장벽을 돌파하여 체내에 침입하면, 선천면역 세포들이 병원체를 감지하여 제거
  - 선천면역 관여: 큰포식세포(대식세포, macrophage), 호중구(neutrophil), 자연살해세포(NK cell), 가지세포(수지상세포, dendritic cell), 보체(complement)
- 이 단계에서 감염이 종식되면 후천면역은 작동하지 않고 면역기억도 남지 않음

**후천면역으로의 전환**
- 선천면역으로 처리할 수 없을 만큼 다량의 병원체 침입 또는 병원체가 선천면역 방어를 돌파했을 때 후천면역 작동
- 감염 부위에서 활성화된 큰포식세포 → **사이토카인(cytokine)** 및 **케모카인(chemokine)** 생산 → 염증반응 유도
- 염증반응으로 활성화된 가지세포가 항원(antigen)을 포식 → 림프절로 이동 → 세포 표면에 항원을 제시하여 후천면역 활성화

**1차 면역반응(primary immune response)**
- 림프구가 가지세포 표면에 제시된 항원을 인식 → 항원 특이적인 T세포와 B세포의 증식과 분화 시작
- 일부 림프구는 면역기억을 가진 **기억세포(memory cell)**로 분화

**2차 면역반응(secondary immune response)**
- 같은 병원체가 재침입하면 기억세포가 신속하게 반응
- 빠르고 강력한 반응 → 재감염으로부터 신속한 숙주 방어

#### 3) 사이토카인(cytokine)과 케모카인(chemokine)

- 면역담당세포는 각종 자극을 받으면 **사이토카인**을 분비 → 면역계의 소통 물질
- **사이토카인**: 세포 간 정보전달 분자로서 작용하여 국소나 전신의 면역계통을 조절
  - 인터페론(interferon, IFN): 바이러스 감염 방어
  - 인터루킨(interleukin, IL): 백혈구의 상호작용에 관여
  - 집락자극인자(colony stimulating factor, CSF)
  - 종양괴사인자(tumor necrosis factor, TNF)
  - 형질전환증식인자(transforming growth factor, TGF)
- **케모카인(chemokine)**: 사이토카인 중에서도 특히 면역담당세포의 체내 이동과 국소 제어를 담당

> **용어 정리**
> - 증식(proliferation): 수와 양이 많아지는 것
> - 분화(differentiation): 어떤 세포가 다른 특징을 갖는 세포로 바뀌는 것

### 2. 면역계통의 구성

#### 1) 면역 담당 세포
- 골수에 존재하는 공통의 전구세포인 **조혈모세포(hematopoietic stem cell)**에서 유래
- 조혈모세포에서 **림프계 전구세포**와 **골수계 전구세포**가 생성

| 전구세포 | 유래 세포 |
|---|---|
| **림프계 전구세포** (5종류) | T세포, B세포, NK세포, NKT세포, 가지세포 |
| **골수계 전구세포** (8종류) | 단핵구, 큰포식세포, 호중구, 호산구, 호염기구, 가지세포, 비만세포, 적혈구·혈소판 등 |

- 면역 담당 세포는 골수 안팎에서 최종 분화하여 각각 고유의 기능을 가진 **작동세포(effector cell)**가 됨

> **주요 세포 영문명**
> - T세포: Thymus cell (가슴샘에서 성숙)
> - B세포: Bone marrow cell (골수에서 성숙)
> - NK세포: Natural killer cell
> - NKT세포: Natural killer T cell
> - 가지세포: Dendritic cell (수지상세포) — 선천·후천 면역 모두에 관여, **항원제시세포(APC, Antigen Presenting Cell)**의 역할

#### 2) 면역 담당 세포의 성숙·분화와 그 기능

**(1) 림프구(lymphocyte)**
- 백혈구의 한 종류, 전체 백혈구 중 약 **25%** 차지
- 크게 B세포, T세포, NK세포로 구분
- B세포는 **골수(bone marrow)**에서, T세포는 **가슴샘(흉선, thymus)**에서 성숙
- T세포와 B세포는 세포 표면에 **항원수용체(antigen receptor)** — T세포수용체(TCR), B세포수용체(BCR) — 가 있어 항원을 인식 → 후천면역의 "항원 특이성"

**B세포의 분화**
- 전구 B세포: B세포수용체가 항원과 반응 → **형질세포(plasma cell)**로 분화하여 **항체** 생산

**T세포의 분화**
- 전구 T세포: 가슴샘에서 **CD4 T세포**와 **CD8 T세포**로 분화

| T세포 종류 | 표면 항원 | 분화 / 기능 |
|---|---|---|
| **CD4 T세포** | CD4 항원 | **보조T세포(helper T cell)**: B세포 활성화(항체 생산 도움), 다른 세포에 작용해 면역·염증 관여 |
| | | **억제T세포(suppressor T cell)**: 자기에 대한 면역반응 방지, 면역계통의 균형 담당 |
| **CD8 T세포** | CD8 항원 | **세포독성T세포(cytotoxic T cell)**: 바이러스 감염 세포, 종양세포, 이식세포 등을 직접 살해 |

**NK세포와 NKT세포**

| 세포 | 특징 |
|---|---|
| **NK세포(natural killer cell)** | 항원 수용체 없음 (선천면역). 종양세포 및 바이러스 감염세포(비정상 자기)를 살해 |
| **NKT세포(natural killer T cell)** | 당지질을 인식. 생체방어와 면역제어 모두에 관여 |

**(2) 탐식세포(phagocyte)** — 선천면역에서 중요
- 미생물 등을 인식해서 탐식(잡아먹기)하는 세포
- 3종류: 큰포식세포, 가지세포, 호중구

| 탐식세포 | 특징 |
|---|---|
| **큰포식세포(macrophage)** | 단핵구에서 분화. 인체 각 조직에 널리 분포하는 장수 세포. 탐식작용 수행 + 사이토카인·케모카인 생산 → 염증반응 유도 |
| **가지세포(dendritic cell)** | 이물질을 흡수·처리하는 능력 + 후천면역 관여 시 세포 표면에 처리한 이물질의 일부분을 제시(**항원제시, antigen presentation**) → 림프구에 항원 인식시킴 |
| **호중구(neutrophil)** | 수명은 짧지만 선천면역 세포 중 **가장 수가 많음**. 체내 침입 미생물을 탐식하여 죽임 |

> **항원제시 과정**: 항원제시세포(큰포식세포, 가지세포)가 세균(비자기)을 포식 → 항원 분해 → **MHC 분자(주조직적합성복합체)**와 결합하여 세포 표면에 항원 조각 제시 → T세포수용체가 이를 인식

**(3) 과립백혈구(granulocyte)** — 다핵백혈구라고도 불림

| 종류 | 염색 특성 | 주요 기능 |
|---|---|---|
| **호중구(neutrophil)** | 중성 색소에 염색 (HE 염색 → 분홍색) | 혈액 내 가장 많은 백혈구. 세균 감염 부위로 **가장 먼저 도착** (염증초기, 급성염증). 탐식작용 |
| **호산구(eosinophil)** | 산성 색소에 염색 (HE 염색 → 밝은 빨강) | 활성화 시 과립 내 물질을 세포 밖으로 방출. **기생충 면역**에 중요한 역할 |
| **호염기구(basophil)** | 염기성 색소에 염색 (HE 염색 → 어두운 파랑) | 활성화 시 과립 내 물질을 세포 밖으로 방출. **비만세포와 함께 알레르기 증상 유발**에 중요 |

> **비만세포(mast cell)**
> - 결합조직 중에 널리 분포하는 세포, 히스타민과 헤파린 등을 함유한 과립을 가진 백혈구의 일종
> - 세포 표면에 IgE 수용체와 결합해 있는 IgE에 항원이 반응하면 활성화
> - 화학매개체(chemical mediator) 방출: 히스타민, 세로토닌, 헤파린, 류코트리엔, 프로스타글란딘
> - 이 화학매개체는 민무늬근(평활근)을 수축시키고 혈관투과성을 높여 **즉시형 알레르기**를 일으킴

### 3. 림프조직

- 선천면역 담당 세포: 전신의 조직에 분포
- 후천면역 관련 림프구: **1차 림프조직**과 **2차 림프조직**이라는 특정 조직에 존재
- 생산된 림프구는 1차 림프조직에서 항원에 비의존적으로 성숙 → 말초조직으로 이동 → 비활성 상태로 혈액과 2차 림프조직 사이를 끊임없이 순환

| 구분 | 역할 | 구성 |
|---|---|---|
| **1차 림프조직 (중추림프조직)** | 림프구가 **생산**되는 장소 | 골수(bone marrow), 가슴샘(흉선, thymus) |
| **2차 림프조직 (말초림프조직)** | 외부 자극에 의한 **면역반응이 일어나는** 장소 | 림프절(lymph node), 지라(비장, spleen), 점막연관림프조직(MALT) |

- 비활성 상태의 림프구는 특수 혈관인 **고내피세정맥(high endothelial venule, HEV)**을 통해 림프절 안으로 이동
- 림프구 표면과 HEV 세포막 위의 상보성 **부착분자(adhesion molecule)**에 의해 특정 림프조직에 머묾
- 그곳에서 림프구는 항원제시세포(APC)를 통해 항원과 만나 순환을 멈추고 활성화

### 4. 선천면역 (innate immunity)

- 비특이적 / 면역기억 없음 / 신속한 반응
- 제인웨이(Janeway)의 정의: 미생물 세포 표면에 존재하는 당사슬의 반복서열을 이물질로 인식하는 면역기억이 없는 시스템
- 감염병 초기에 선천면역이 작용하여 대부분의 병원성 미생물 제거
- 선천면역에서 제거되지 못한 병원성 미생물 → 체내 증식 시작 → 선천면역과 연동하여 후천면역 작동

> **선천면역 → 후천면역 연결 비유**
> 가지세포(관할지 순찰 경찰관)가 감염부위에서 항원 포식(범인 현장검거) → 림프절로 이동(관할 경찰서로 이동) → 후천면역의 림프구에게 항원 제시(범인 정보를 보고) → 특이적 항원 인식하는 후천면역으로 연결(범인 소속 특정 범죄집단의 빠른 소탕)

#### 1) 물리적 방어인자

| 방어인자 | 기전 |
|---|---|
| **건조한 피부 표면** | 세균이나 진균 등의 미생물은 일반적으로 증식 불가능 |
| **체내의 점막상피조직** | 뮤신(mucin) 등의 단백질을 함유하는 점액이 표면을 덮어 미생물의 직접 접촉이나 상피 손상을 억제. 점막은 1층의 상피조직으로 이루어져 손상을 입기 쉬워 미생물 감염에 매우 중요한 침입구 |
| **뮤신 등의 점액** | 단독으로 바이러스 등과 결합하여 응집시키는 감염방어작용 |
| **호흡계통의 기도상피** | 기도 점막상피의 **섬모운동**을 통해 이물질 제거 |
| **창자 등의 꿈틀운동(연동운동)** | 미생물의 국소 정체나 비정상적 증식을 막음 |
| **창자·질 점막상피의 정상균무리** | **정상균무리(normal flora)**가 생물학적 장벽 형성. 창자에서는 무산소성 환경(장내세균), 질에서는 산성 환경(유산균)을 만들어 병원성 세균의 증식 억제 |

**선천면역의 방어인자 분류**

| 세포성 방어인자 | 체액성 방어인자 |
|---|---|
| 호중구, 단핵구, 큰포식세포, NK세포, NKT세포 등 | 보체, 콜렉틴, 리소자임, 펩신, 항균펩티드, 락토페린, 트랜스페린, 단백질분해효소, 지방산, 사이토카인, 인터페론 등 |

#### 2) 체액성 방어인자

**(1) 보체(complement)** — C1~C9까지 존재 (발견 순서대로 명명)
- 항체의 작용을 보조하는 혈청단백질로서 발견됨

**보체의 생체방어 역할 3가지**

| 역할 | 내용 |
|---|---|
| **① 옵소닌화(opsonization)** | 침입한 미생물이나 이물질을 옵소닌화 → 탐식세포가 이물질이나 면역복합체(항원항체보체복합체)를 탐식하여 파괴 |
| **② 백혈구 동원 및 활성화** | 보체활성화 과정에서 생성되는 저분자 보체성분(C5a, C3a, C4a 등)이 백혈구를 국소 부위로 동원하고 활성화 → **아나필락시스 독소** |
| **③ 막공격복합체(MAC) 형성** | 보체활성화에 동반하여 **막공격복합체(membrane attack complex, MAC)** 형성 → 미생물이나 이물질에 직접 구멍을 뚫어 공격·파괴 |

**보체의 3가지 역할 상세**

**(1) 옵소닌화와 탐식 촉진**
- 옵소닌화(opsonization): 침입한 미생물에 보체가 결합 → 탐식세포의 보체수용체를 통해 탐식기능 촉진 → 미생물 파괴
- 활성화한 보체성분과 항체가 협조하여 옵소닌화를 이룸
- 혈중 가장 많이 존재하는 **C3**이 활성화하여 생성되는 분해물 **C3b**가 특히 중요한 역할
- 보체 명명 규칙: C3 → 작은 조각 **C3a** (아나필락시스 독소), 큰 조각 **C3b** (옵소닌화 도움)
- C3b 분자가 세균이나 항원항체복합체에 결합 → 탐식세포에 대한 결합성을 높여 탐식 촉진

**(2) 저분자 보체성분에 의한 백혈구의 국소 동원과 활성화**
- 보체성분 C3, C4, C5는 활성화 과정에서 저분자 보체성분 **C3a, C4a, C5a**를 생성
- C3a와 C5a는 호중구나 큰포식세포에 대해 강력한 유주 활성(이동 활발화)
- 혈관내피세포나 호중구에 작용하여 접착성을 높이고, 국소에 탐식세포가 모이도록 함
- 방출된 염증전달물질 → 혈관투과성 항진 + 민무늬근(평활근) 수축 촉진 → **아나필락시스독소(anaphylatoxin)**
- 생리활성 강도 순서: **C5a > C3a > C4a**

**(3) 막공격복합체(MAC) 형성에 의한 세포·세균의 파괴 및 용해**
- 보체활성화 후기반응으로서 MAC라는 막관통형의 고리 유사 분자가 형성
- 강력한 막 공격성을 나타내어 미생물이나 이물질의 세포막에 직접 구멍을 뚫어 파괴

**보체활성화의 3가지 경로**

| 경로 | 시작 | 과정 |
|---|---|---|
| **고전경로 (classical pathway)** | **IgM 또는 IgG**가 미생물에 결합 → 항체의 Fc 부위에 **C1**이 부착·활성화 | C1이 C4와 C2를 절단 → C4b2a 복합체(C3 전환효소) 형성 → C3 분해 → C3b 생성 → C4b2aC3b(C5 전환효소) 형성 |
| **렉틴경로 (lectin pathway)** | 항체 없이 혈장의 **만노스 결합 렉틴(MBL)**이 미생물에 부착하여 활성화 | MBL에 결합한 효소가 C4와 C2를 절단 → C4b2a(C3 전환효소) → C3 분해 → C4b2aC3b(C5 전환효소) |
| **대체경로 (alternative pathway)** | C3의 **자연 가수분해**산물인 C3b가 미생물 표면에 부착 | C3b + Bb(factor B가 factor D에 의해 분해) → C3bBb(C3 전환효소) → C3 분해 증폭 → C3bBbC3b(C5 전환효소) |

> **보체 반응 핵심인자: C3b**
>
> **보체의 효과/작용 결과 정리**
> 1. 옵소닌화 (C3b, C5b) → 탐식 촉진
> 2. 아나필락톡신 (C3a, C5a) → 염증반응
> 3. 막공격복합체 (C5b, C6, C7, C8, C9) 형성 → 미생물 직접 파괴

**(2) 항균물질 등**
- **항균펩티드(antimicrobial peptide)**: 창자·기도 상피나 피부에 있는 디펜신(defensin), 카텔리시딘(cathelicidin) 등이 세균의 세포막을 직접 파괴하여 살균
- **콜렉틴**: 허파꽈리상피에서 분비되는 표면 활성단백질 A, D가 미생물과 결합 → 옵소닌화 → 응집 등을 일으켜 이물질 제거 촉진
- **리소자임(lysozyme)**: 침, 땀, 눈물에 존재하는 항균효소 — 미생물에 대한 화학적 장벽
- **위산, 펩신(pepsin)** 등의 소화효소: 화학적 장벽 역할
- **락토페린(lactoferrin)** / **트랜스페린(transferrin)**: 모유나 혈액 속에 존재. 세균의 유리철(Fe³⁺ 이온) 이용을 저해 → 증식 억제 → **정균작용(bacteriostatic action)**

**(3) 인터페론(interferon, IFN)**
- 바이러스나 세균이 숙주 세포와 만나서 발현되는 단백질
- **바이러스간섭(viral interference)**: 세포 수준에서 바이러스 감염이 일어나면 다른 바이러스의 증식을 저해하는 현상 → 인터페론이 관여

| 종류 | 역할 |
|---|---|
| **I형 IFN (IFN-α/β)** | 바이러스간섭작용 담당. 바이러스 감염 → I형 IFN 합성 및 분비 → 주위 세포의 표면 수용체와 결합 → 새로운 유전자 발현 유도 → 바이러스 증식 억제 |
| **II형 IFN (IFN-γ)** | 면역세포 간 중개인자로 작용 |

#### 3) 세포성 방어인자

**(1) 탐식세포(phagocyte)**
- 미생물이 체내로 침입하면 선천면역계의 탐식세포가 작용
- 호중구(감염 부위에 1등으로 출동)와 큰포식세포(대식세포)가 중심
- 미생물 인식 방법: 직접 수용체를 통한 인식 또는 보체·항체에 의해 옵소닌화된 미생물을 Fc수용체나 보체수용체를 통해 흡수·탐식

| 세포 | 특징 |
|---|---|
| **호중구(neutrophil)** | 세포질에 많은 과립을 가진 수명이 짧은 백혈구. 혈류에 많이 존재하며 정상 조직에는 별로 없음 |
| **큰포식세포(macrophage)** | 혈관을 순환하는 단핵구가 혈관 밖의 체내조직으로 이동하여 활성화된 것 |

> **큰포식세포의 조직별 이름**
> - 허파(폐): 먼지세포
> - 간: 쿠퍼세포
> - 뇌: 미세아교세포
> - 지라(비장)나 소화관의 결합조직에도 존재

**탐식세포의 탐식 및 살균기전**
1. 탐식세포가 수용체를 통해 미생물과 결합
2. 세포 내부에 신호 전달
3. 세포뼈대를 변화시켜 세포막이 미생물을 둘러싸 **포식소체(phagosome)** 형성
4. 포식소체는 항균물질(디펜신, 골수세포형과산화효소(MPO), 리소자임, 락토페린 등)을 함유하는 여러 개의 **용해소체(lysosome)**와 융합 → **포식용해소체(phagolysosome)** 형성
5. 미생물을 파괴 및 분해

**주요 면역 담당 세포의 표면항원과 수용체**

| | αβT세포 | γδT세포 | B세포 | NK세포 | 큰포식세포 | 호중구 |
|---|---|---|---|---|---|---|
| 항원수용체 | TCR(αβ) | TCR(γδ) | 표면 IgM | (불명) | − | − |
| Fc수용체 | − | − | −, + | −, + | + | + |
| 보체수용체 CR1 | − | − | + | − | + | + |
| 보체수용체 CR2 | − | − | + | − | − | − |
| 보체수용체 CR3 | − | − | − | + | + | + |
| 주요 CD항원 | CD3, CD4, CD8 | CD3, (CD4⁻), (CD8⁻) | CD19, CD20, CD21 | CD56, CD57 | CD11b, CD14 | CD13, CD17, CD66 |
| 면역계통 | 후천면역 | 선천면역 | 후천면역 | 선천면역 | 선천면역 | 선천면역 |

**(2) NK세포, NKT세포**

| 세포 | 특징 |
|---|---|
| **NK세포 (자연살해세포)** | 종양세포(비정상 자기세포)를 파괴하는 T세포 계열. T세포수용체를 갖지 않고 가슴샘에서 분화하지 않음 (선천면역). 표면에 활성화수용체와 억제수용체를 가지며 수용체를 통한 신호의 균형에 따라 손상 여부 결정. 인플루엔자바이러스, 센다이바이러스 등의 바이러스 감염 세포를 손상. NK세포 결핍 시 HSV-1 감염 저항성이 현저하게 저하 |
| **NKT세포 (자연살해T세포)** | NK세포의 성질과 T세포수용체를 갖는 세포 무리. 결핵균 감염 세포의 **CD1d**에 제시된 **당지질항원**을 인식 |

### 5. 후천면역 (acquired immunity)

- 항원 특이적 / 면역기억 / 느리지만 강력함
- B세포와 T세포가 중심 역할, 각각 세포 표면에 항원 특이적 항원수용체를 발현
  - **B세포수용체(BCR)** = 항체(antibody) = 면역글로불린(immunoglobulin, Ig)
  - **T세포수용체(TCR)**

**후천면역의 특징**

| 특징 | 의의 |
|---|---|
| 항원 특이성 | 특정 항원에 대하여 생긴 면역은 특정 항원에만 면역능력을 나타냄 |
| 다양성 | 숙주 자신의 물질을 제외한 다양한 항원에 대해 면역반응이 일어남 |
| 면역허용 | 숙주 자신의 물질에 대해서는 면역반응을 일으키지 않음 |
| 항원 특이적 클론확장 | 항원을 인지할 수 있는 림프구만 선택적으로 활성화 |
| 면역기억 | 처음 항원과 접촉한 후 기억세포가 형성되면 동일한 항원에 재노출되었을 때 더 신속하고 강력한 면역반응이 일어남 |

#### 1) 항원수용체 및 항체의 구조와 기능

- BCR과 TCR의 공통 구조적 구성:
  - **가변부위(V부위, variable region)**: 다양한 항원과 결합할 수 있도록 구조적 다양성이 풍부
  - **불변부위(C부위, constant region)**: 아미노산 배열이 일정

**① B세포수용체(BCR)**
- 세포 표면에 발현하는 항체
- 2개의 **무거운 사슬(H사슬, heavy chain)** + 2개의 **가벼운 사슬(L사슬, light chain)** → 4개의 폴리펩티드사슬이 디설피드결합으로 1분자 구성
- H사슬의 막관통부위에서 세포막 표면과 결합
- L사슬: κ(제2번 염색체)와 λ(제22번 염색체) 2개의 유전자자리로 구성
- 동일한 가변부위(V부위) 2개를 가지며 항원의 **항원결정인자(epitope)**를 인식하여 결합
- 세포 내 신호전달을 위해 **Igα/Igβ**와 복합체를 형성 → B세포의 활성화와 분화에 중요

**② 항체의 클래스 (GAMDA)**
- 항체는 V부위와 C부위를 가지며 막결합부위가 없는 단백질로서 분비
- C부위의 차이에 따라 **5종류의 클래스**: IgG, IgA, IgM, IgD, IgE
- 하나의 성숙 B세포는 IgM과 IgD를 발현하지만, IgM만 분비하고 IgD는 분비하지 않음
- 분비형 IgM은 **J사슬**과 연합하여 **오량체**, IgA는 J사슬과 연합하여 **이량체** 형성

**항체의 클래스와 서브클래스**

| 클래스 | IgM | IgD | IgG | IgA | IgE |
|---|---|---|---|---|---|
| 구조 | 오량체 (J사슬) | 단량체 | 단량체 | 이량체 또는 단량체 (J사슬, 분비성분SC) | 단량체 |
| 분자량 | 900,000 | 180,000 | 150,000 | 390,000 (단량체 170,000) | 190,000 |
| 구성사슬 | μ, κ, λ, J | δ, κ, λ | γ, κ, λ | 이량체: α, κ, λ, J, SC | ε, κ, λ |
| 서브클래스 | − | − | IgG1, IgG2, IgG3, IgG4 | IgA1, IgA2 | − |
| H사슬 동형 | μ | δ | γ1, γ2, γ3, γ4 | α1, α2 | ε |
| 기능 | 비활성 B세포의 BCR, 보체활성화능력 높음, 중화활성 | 비활성 B세포의 BCR | 보체활성화/옵소닌화/항체의존성 세포손상 활성은 IgG1·IgG3에서 높음, 중화활성 모든 서브클래스에서 유효, **태반 이행성** 높음 | 이량체는 상피 분비성분과 만나 분비형이 됨, **결막면역**, 모유 이행성, 중화활성 | **비만세포**, 기생충 감염에 반응, **탈과립** |
| 혈청치 (성인, mg/mL) | 1.5 | 0.03 | 9 (IgG1) | 3.0 (단량체 IgA1) | 0.00005 |
| 수용체 | − | − | FcγR (IgG1, IgG3) | FcαR (단량체만) | FcεR (비만세포) |
| 반감기 (일) | 10 | 3 | 20~21 (IgG3은 7일) | 6 | 2 |

**③ T세포수용체(TCR)**
- **α사슬/β사슬**로 구성되는 분자와 **γ사슬/δ사슬**로 구성되는 분자 → 각각 **αβT세포**, **γδT세포**
- 주집단인 αβT세포: 1종류의 α사슬 + 1종류의 β사슬 → 두 사슬에서 1개의 항원인식부위 형성
- T세포수용체는 신호분자로서 4종류의 폴리펩티드사슬(γ, δ, ε, ζ)로 구성되는 **CD3 복합체**와 만나 T세포의 분화나 활성화에 작용

**④ 항체의 기능 4가지**

| 기능 | 내용 |
|---|---|
| **중화** | 미생물이나 미생물독소와 결합하여 숙주세포 부착을 억제 → 감염 및 작용 차단 |
| **옵소닌화 및 탐식 촉진** | 미생물 표면에 결합(도포) → 항체수용체를 가진 탐식세포의 포식작용 촉진 |
| **항체의존세포독성 (ADCC)** | 항체가 비정상세포에 부착 → 항체수용체를 지닌 NK세포가 항체와 결합·활성화 → 항체로 도포된 비정상세포를 살해 |
| **보체 활성화 (고전경로)** | 미생물과 결합한 항체가 고전경로를 통해 보체 활성화 → 염증반응, 포식작용 촉진, 미생물 용해 유도 |

#### 2) 항원의 특징

**(1) B세포수용체와 항체가 인식하는 항원**
- BCR은 단백질, 당사슬, 지질, 핵산 등의 고분자물질에서부터 펩티드, 저분자 화학물질에 이르기까지 다양한 입체구조를 **항원결정인자(epitope)**로서 인식

**(2) T세포수용체가 인식하는 항원**
- T세포에는 자기와 비자기를 식별하는 기능이 있으며, 이는 항원의 구조적 특징에 기인
- **주조직적합복합체(MHC, major histocompatibility complex)**: 사람의 경우 **사람백혈구항원(HLA, human leukocyte antigen)**이라 불림
- MHC는 세포 내의 단백질항원이 분해되어 생긴 펩티드와 결합하여 이를 T세포에 제시

| MHC 클래스 | 구성 | 결합 펩티드 |
|---|---|---|
| **클래스 I** | 클래스 I α사슬 + β2저분자글로불린 | **8~9개**의 펩티드와 결합 |
| **클래스 II** | 클래스 II α사슬 + β사슬 | **10~30개** (평균 15개 전후)의 펩티드와 결합 |

**⑤ T세포의 MHC 인식**
- **세포독성T세포(CTL)**: CD8을 발현 → T세포수용체 + CD8이 **MHC 클래스 I** 분자와 그것에 결합한 펩티드를 인식
- **보조T세포(Th세포)**: CD4를 발현 → T세포수용체 + CD4가 **MHC 클래스 II**와 그것에 결합한 펩티드를 인식
- MHC 분자는 **다형성(polymorphism)**을 띠며, 자기와 다른 유전형 MHC 분자를 발현하는 세포는 비자기로서 CTL에 의해 파괴됨 → 장기이식 거부반응의 기전

#### 3) MHC에 의한 항원제시기전

**(1) MHC 클래스 I 경로 (내재성 항원 제시)**
- MHC 클래스 I은 **거의 모든 유핵세포**에서 발현
- 핵 내부나 세포질에 있는 단백질 → **프로테아좀(proteasome)**에 의해 분해 → **세포질그물(소포체)**에 흡수 → MHC 클래스 I과 결합 → 세포막 표면으로 운반
- 자기항원이나 **바이러스 감염 세포의 바이러스항원**이 이 경로에서 제시됨 (경로 ①)

**(2) MHC 클래스 II 경로 (외래·탐식항원 제시)**
- MHC 클래스 II는 **가지세포, B세포, 큰포식세포** 등 **항원제시세포(APC)**에서 발현
- 탐식된 외래항원 → **엔도솜(endosome)** 내에서 분해 → MHC 클래스 II와 결합 → 세포막 표면으로 제시 (경로 ②)
- 항원제시세포는 MHC 클래스 I과 클래스 II를 모두 발현하여 CTL과 Th세포 **모두와 상호작용** 가능

**(3) 교차제시(cross-presentation) — 가지세포 특유**
- 가지세포에는 탐식된 **외래항원을 MHC 클래스 I에 제시**할 수 있는 특수 경로가 존재
- **경로 ③**: 외래항원이 엔도솜에서 세포질로 운반 → 프로테아좀 분해 → 세포질그물 내로 흡수 → MHC 클래스 I과 결합
- **경로 ④**: 엔도솜 내에서 직접 MHC 클래스 I과 결합

> **항원제시 정리**
>
> | 경로 | 항원 유래 | MHC 클래스 | 인식하는 T세포 | 발현 세포 |
> |---|---|---|---|---|
> | 내재성 경로 (①) | 세포질 내 단백질 (자기/바이러스) | 클래스 I | CTL (CD8⁺) | 거의 모든 유핵세포 |
> | 외래항원 경로 (②) | 탐식된 외래항원 | 클래스 II | Th세포 (CD4⁺) | APC (가지세포, B세포, 큰포식세포) |
> | 교차제시 (③④) | 탐식된 외래항원 | 클래스 I | CTL (CD8⁺) | 가지세포 |

#### 4) 면역기억 (immunologic memory) — 예방접종의 원리 ★★★

- **면역기억**: 면역계통이 이전에 접촉한 항원에 2차로 노출되었을 때 **더 신속하고 강력하게** 반응하는 능력
- 외래항원에 처음 노출 → **1차 면역반응** 발생
- 1차 면역반응 중 형질세포에서 분비된 항체와 소수의 **기억B세포(memory B cell)** 및 **기억T세포(memory T cell)**가 체내에 남아 면역기억 형성
- 기억세포는 장기간 생존하며 **2차 림프조직**에서 대기 → 동일한 항원 재노출 시 **2차 면역반응** → 항원 신속·강력 제거

> **1차 면역반응 vs 2차 면역반응**
>
> | | 1차 면역반응 | 2차 면역반응 |
> |---|---|---|
> | 반응 속도 | 느림 | **신속** |
> | 항체 농도 | 낮음 | **높음** |
> | 반응 강도 | 보통 | **강력** |
> | 기억세포 | 형성됨 | 기억세포가 활성화 |

### 6. 후천면역의 상호작용과 감염반응

#### 1) 후천면역과 선천면역의 상호작용

**(1) 전신면역 — 림프절에서의 면역반응**

- 혈류를 통해 림프절에 도달한 T세포 → **곁겉질(곁피질, paracortex)**에 분포 (①)
- B세포 → **림프소절(follicle)**에 분포 (②)
- 항원(③)은 **들림프관(수입림프관)**을 통해 림프절로 유입
- **가지세포**(④)가 CTL 및 Th세포에 항원 제시
- 림프소절(1차 림프소절)의 B세포 → 항원 인식하여 활성화 + Th세포로부터 **CD40 자극** 수용 (⑤)
- B세포 → **배중심(germinal center; 2차 림프소절)** 형성 → 활성화·성숙 진행 → 림프소절 가지세포 및 사이토카인 영향 → **형질세포로 분화** (⑥)
- 생산된 항체, 활성화 CTL, Th세포 → **날림프관(수출림프관)** 통해 혈류·체순환으로 진입 → 감염 병소로 이동 (⑦)

**(2) 점막면역 — MALT에서의 면역반응**

- 점막에서는 창자(집합림프소절), 편도, 기관지 등에 존재하는 **점막연관림프조직(MALT, mucosa-associated lymphoid tissue)**에서 면역 유도
- MALT에는 들림프관이 없음 → 외래항원은 점막층 틈새의 **M세포(microfold cell)**를 통해 MALT로 흡수 (⑧)
- MALT에 상주하는 Th세포, B세포, 가지세포, 큰포식세포 등이 흡수된 항원에 즉시 반응 (⑨)
- 활성화된 B세포 → 림프소절로 이동 → **IgA로 클래스전환(class switch)** 유도 (⑩)
- 혈류·체순환(⑪)을 거친 후 점막조직(⑫)으로 귀환 → B세포가 형질세포로 분화
- IgA는 상피세포 수용체와 결합 → 속공간으로 운반 → 절단되어 **분비성분(SC)**과 함께 분비 (⑬)
- 점막에서 IgA의 역할: **중화작용**으로 세균 침입 저지 + **모유로 이행** → 신생아 구강 점막 초기 면역에 중요

#### 2) 후천면역의 감염반응

**(1) 세균 및 기생충 감염반응**

| 감염 유형 | 면역반응 기전 |
|---|---|
| **세포 내 기생세균/원충** | MHC 클래스 I에 제시된 세포 내 항원 → **CTL**에 의한 제거 |
| **엔도솜 내 기생세균** (큰포식세포 등) | MHC 클래스 II에 세균항원 제시 → **Th세포** 반응. 특히 **Th1세포의 IFN-γ** → 큰포식세포 활성화 + 식포 내 살균작용 증강 |
| **세포벽/당사슬로 보호되는 세균** | 보체 의존적 손상은 어렵고, **옵소닌화 + 탐식세포 처리**가 유효 |
| **연충(helminth)** | **IgE 항체**가 많이 유도 → **호산구 등에 의한 탈과립반응**이 효과적 |

**(2) 바이러스 감염반응**

- 바이러스 감염 세포의 제거:
  - MHC 클래스 I에 바이러스항원 제시 → **CTL 반응**
  - 항체를 통한 **보체의 막공격복합체(MAC)** 형성
  - **NK세포의 항체의존세포독성(ADCC)**
- **IFN-α/β(I형)**: 상피 포함 각종 세포가 생산 → 세포 내 바이러스 증식 억제 + MHC 분자 발현 증강 → CTL 인식 효율 향상
- NK세포: IFN-α/β에 반응하여 활성화
- **CTL 및 Th1세포**: 항원제시에 반응하여 **IFN-γ(II형)** 생산 → 항원제시능력 증강 + 큰포식세포 활성화 → 바이러스입자 및 감염 세포 대처

### Advance: 이물질제거기전

#### 1) 항체에 의한 이물질제거 — 5가지 기전

| 기전 | 내용 | 대상 병원체 |
|---|---|---|
| **① 보체활성화** | 이물질 + 항체 → 보체 활성화 → 막공격복합체 형성 또는 iC3b 침착(옵소닌화) 촉진 | 세포 외 기생세균, 원충, 바이러스 |
| **② 탐식 촉진 (옵소닌화)** | 이물질을 표식한 항체 → Fc수용체(FcR)를 통해 탐식세포와 호중구 탐식 및 살균 촉진 | 세포 외 기생세균, 원충 |
| **③ 항체의존세포독성 (ADCC)** | 항체를 표적으로 NK세포가 FcR을 통해 ADCC 촉진 → 퍼포린·그랜자임 분비 | 바이러스 |
| **④ 중화활성** | 항원의 기능부위를 덮어 접착이나 모임 억제 (중화) | 세균, 원충(전반), 바이러스 |
| **⑤ 탈과립반응** | 기생충을 인식하는 IgE → Fcε수용체(FcεR)를 가진 과립백혈구(호산구, 비만세포, 호염기구)의 탈과립반응 유도 | **연충** |

#### 2) T세포에 의한 이물질제거

| T세포 유형 | 기전 | 대상 병원체 |
|---|---|---|
| **① CTL** | 비자기 항원을 MHC 클래스 I에 제시한 세포 인식 → 퍼포린·그랜자임 → 세포자멸사 유도 (또는 TNF-α, Fas 리간드-Fas 경로) | 세포 내 기생세균, 원충, 바이러스 |
| **② Th1** | MHC 클래스 II 항원제시에 반응 → CTL 활성화, **IFN-γ 생산** → NK세포 활성화, 큰포식세포 살균력 증강 | 세포 내 기생세균, 원충 |
| **③ Th2** | IL-4, IL-5, IL-13 생산 → 호산구, 호염기구, 비만세포 동원·활성화, B세포에 작용하여 **IgE/중화항체** 생산 촉진 | 기생충 감염 |
| **④ Th17** | IL-17 생산 → 상피세포·섬유모세포에 작용 → 케모카인 유도 → **호중구 동원**. 과도한 반응은 자가면역 조직손상이나 염증성 장질환에 관여 | 사이토카인·케모카인에 의한 간접적 감염 방어 |
| **⑤ TFH (소포성 보조T세포)** | 림프소포에 존재하며 **B세포와 상호작용** → 항체 생산 촉진 | 사이토카인·케모카인에 의한 간접적 감염 방어 |

### 7. 면역계통의 병리와 응용

#### 1) 면역결핍증(immunodeficiency disease)

- 면역세포와 분자가 정상적으로 분화·증식해야 면역반응이 이루어짐
- 어느 하나라도 결핍되면 다양한 면역결핍증 발생
- **원발성 면역결핍증**: 유전적 원인 (생후부터 발현)
- **후천성 면역결핍증**: 바이러스 감염 등 (HIV 등)

**(1) 원발성 면역결핍증** — 150종류 이상

- 면역계통 구성성분(세포, 분자)의 선천적 결핍/이상 → 면역기능 저하 → 감염병·자가면역에 대한 감수성 증가 → 생후부터 심각한 감염병 반복

**(2) 후천성 면역결핍증**

- **후천면역결핍증후군(AIDS)**: 사람면역결핍바이러스(**HIV**)가 **CD4⁺ T세포** 등에 감염 → MHC 클래스 II 발현 저해 → 보조T세포 기능 유지 불가
- CD4⁺ T세포 숫자 감소 + 기능 저하 → **결핵, 진균 폐렴** 등의 **기회감염** 발생, 종양 발생 가능
- 기타 후천성 면역결핍 원인: 바이러스 감염에 의한 일시적 세포면역 억제, 영양불량, 방사선 피폭, 지라(비장) 적출, 각종 면역억제제 투여

#### 2) 과민반응(hypersensitivity) = 알레르기(allergy) ★★★

- 비교적 무해한 항원에도 개인·환경에 따라 후천면역 활성화가 강력하게 촉진되는 경우가 있음
- 해당 항원에 재노출 시 면역반응이 야기되어 **조직손상**이 일어나는 반응 = 과민반응(알레르기)
- 알레르기 발단 항원 = **알레르기항원(allergen)**
- 후천면역 담당 B세포·항체·T세포가 관여 → 선천면역 담당 큰포식세포·호중구·보체계가 **작동세포(effector cell)**로서 조직손상 실행
- **항체 관여**: I~III형 과민반응 / **T세포 관여**: IV형 과민반응

**과민반응의 분류 (표 2-5)**

| | **I형** | **II형** | **III형** | **IV형** |
|---|---|---|---|---|
| **명칭** | 즉시형 과민반응 | 세포용해성(세포독성)과민반응 | 면역복합체반응 | 지연형 과민반응 |
| **관여 면역반응** | **IgE** | IgM, IgG | IgM, IgG | **감작 T세포** |
| **조직손상의 작동세포** | 비만세포, 호염기구, 호산구 | 보체, NK세포, 호중구, 큰포식세포 | 보체, 호중구, 큰포식세포 | 활성화 큰포식세포, 세포독성T세포 |
| **주요 질환** | 알레르기성 비염, 기관지천식, 두드러기, **아나필락시스** | 용혈빈혈, 혈소판감소 자색반 | 급성사구체신염, 류마티스관절염, 혈청병, 토리콩팥염, 크리오글로불린혈증, **전신홍반루푸스(SLE)** | **이식편대숙주질환(GVHD)**, 접촉성 피부염, 알레르기성 뇌척수염, 염증성 장질환, 결핵성 육아종 |
| **발병기전** | 알레르겐 → IgE가 비만세포의 FcεR에 결합 → **화학매개체 방출** | 표적세포에 IgG 결합 → 보체 활성화 → **세포용해** | 면역복합체(IC)가 혈관내피에 침착 → FcγR + 보체 → **혈관염·조직손상** | CD4 Th1세포 → 큰포식세포 활성화 → 염증반응·세포손상 / CD8 CTL → 표적세포 직접 손상 |

> **FcεR**: IgE가 결합하는 Fc수용체 / **FcγR**: IgG가 결합하는 Fc수용체 / **IC**: 면역복합체 / **Mφ**: 큰포식세포

**I형 과민반응 = 즉시형 과민반응(immediate hypersensitivity) 상세**

- 알레르기항원(allergen)과 결합한 **IgE**가 조직의 비만세포·호염기구 표면 **IgE수용체(FcεR)**와 강력하게 결합
- 동일 알레르겐 재노출 시 → 세포 활성화 및 **탈과립** → 히스타민, 류코트리엔, 케모카인 등 **화학매개체(chemical mediator)** 방출
- **즉시형 반응**: 히스타민이 기관지 민무늬근·혈관내피세포의 **H1수용체**를 통해 기도 수축, 점액 분비, 부종·발적 유발
- **지연형 반응**: 류코트리엔·프로스타글란딘이 주체 → 염증, 민무늬근 수축, 기도 연축, 점막상피 점액분비
- 증상은 반응이 일어나는 **장소에 의존** (국소적 과민반응)
  - 흡입성 알레르겐(꽃가루, 집먼지): 알레르기성 비염, 인두후두부종, 기관지천식
  - 식이성 알레르겐(계란, 밀가루, 우유): 구토, 설사
  - **전신성 알레르기반응**: 아나필락시스(anaphylaxis) → 호흡·순환부전 → 사망 위험

**II형 과민반응 = 세포용해성(세포독성)과민반응 상세**

- **IgG·IgM**과 결합한 표적항원이 조직이나 세포 **표면**에 존재 → 보체의 **고전경로** 활성화 → 해당 조직·세포 손상
- 대표적 손상기전: 적혈구·혈소판 표면에 항체 결합 + 보체 활성화 → 혈구 파괴 → **용혈빈혈, 혈소판감소증**
- 기타: 혈액형 부적합수혈, 신생아 용혈빈혈, 약물성·자가면역용혈빈혈
- **V형 과민반응** (II형의 아형): 표적항원이 조직세포 위 **수용체**인 경우
  - 수용체에 항체 결합 → **작용제(agonist; 활성화)** 또는 **대항제(antagonist; 억제)**로서 작용 → 세포 내 신호 활성화/억제
  - 예: **그레이브스병(Graves disease)** — TSH수용체에 자극 항체 → 갑상샘항진증 / **중증근무력증(myasthenia gravis)** — 아세틸콜린수용체에 차단 항체 → 근력저하

**III형 과민반응 = 면역복합체반응 상세**

- 표적조직과 관계없는 **가용성 항원**이 IgG와 결합 → **항원항체복합체(immune complex, IC)** 형성
- IC가 온몸을 순환하면서 관절공간, 콩팥 토리(사구체), 피부 등의 **혈관벽·바닥막** 조직에 침착
- 보체의 **고전경로** 활성화 → C3a, C4a, C5a 등 **아나필락시스독소** 생성 → 비만세포·호염기구에서 화학매개체 방출 촉진
- 대표 질환: **혈청병, 전신홍반루푸스(SLE)**

**IV형 과민반응 = 지연형 과민반응(delayed-type hypersensitivity, DTH) 상세**

- 감작된 항원에 대해 **T세포**가 항원 특이적으로 작동하는 반응
- 항원 접촉 **1~3일 후** 나타남 → 지연형
- **CD4⁺ Th1세포** 활성화 → **CD8⁺ CTL**이 직접 세포 손상 유발, 또는 분비된 **IFN-γ**에 의해 큰포식세포 활성화 → **육아종(granuloma)** 형성 → 조직손상
- 관련 질환: **투베르쿨린반응**, 접촉피부염, 결핵에 의한 육아종 형성, 다발경화증

#### 3) 자가면역질환(autoimmune disease)

**(1) 자가면역질환의 병태**

- 자기 조직·세포 등 자기항원에 대한 면역반응은 보통 **면역관용(immunologic tolerance)** 상태에 있음
- 어떤 원인으로 이 균형이 무너지면 자기항원을 공격·제거하려는 강력한 후천면역반응이 성립
- **II형, III형, IV형 과민반응**이 해당 → 자기성분에 대한 과민반응
- 발병에는 **유전적 다형성**과 **환경인자**가 깊이 관여
  - **HLA 클래스 I, 클래스 II의 유전적 다형성**이 자가면역질환의 유전소인 대부분을 차지
  - 환경인자: 세균 감염이 계기가 되는 경우가 많음 (예: 고름사슬알균에 의한 류마티스열)
  - 감염으로 인해 변형된 자기가 병원체와 숙주 항원의 유사성을 나타내는 **분자모방(molecular mimicry)**, 또는 항원결정인자에 새로운 항체가 생산됨으로써 자가면역질환 발병

> **과민반응·자가면역질환 흐름 정리**
>
> - **과민반응**: 외래 항원에 대한 과도한 면역반응
> - **자가면역질환**: 자기 유래 또는 변형된 자기가 표적항원이 되는 면역반응
> - 두 경우 모두 후천면역에 기인하는 선천면역의 조직손상이 본태
>   - I형 (즉시형): B세포 → IgE → 비만세포·호염기구·호산구 → 화학매개체
>   - II·III형 (V형 포함): B세포/Th2세포 → IgG/IgM → 호중구·NK세포·보체 → 보체활성화·세포용해
>   - IV형 (지연형): Th1세포/CTL → 큰포식세포·NK세포 → 큰포식세포 활성화·사이토카인에 의한 염증반응

**(2) 자가면역질환의 종류**

**장기 특이적 자가면역질환** — 특정 장기에만 자가면역현상이 일어남 (세포 표면·세포바깥기질 유래 항원에 대한 항체반응 = II형 과민반응)

| 굿파스처증후군 | 콩팥, 허파의 바닥막단백질 | 토리콩팥염, 허파꽈리출혈 | |
| 보통천포창 | 표피 카드헤린 | 피부물집 | |
| 급성 류마티스열 | 사슬알균 항원, 심장세포와의 교차항체 | 관절염, 심근염, 심장관막 흉터형성 | |
| **그레이브스병** | TSH수용체 | **갑상샘항진증** | **V형** |
| **중증근무력증** | 아세틸콜린수용체 | **진행성 근력저하** | **V형** |

**장기 비특이적(전신성) 자가면역질환**

| 과민반응 유형 | 자가면역질환 | 대응하는 자가항원 | 주요 임상증상 | 비고 |
|---|---|---|---|---|
| 면역복합체질환 (III형) | 전신홍반루푸스(SLE) | DNA, 히스톤, 리보솜, snRNP, scRNP | 토리콩팥염, 혈관염, 관절염 | 전신성 병태 |
| | 아급성 세균심내막염 | 세균 항원 | 토리콩팥염 | 전신성 병태 |
| | 본태성 크리오글로불린혈증 | 류마티스인자, IgG복합체 | 전신성 혈관염 | 전신성 병태 |
| T세포 통한 질환 (IV형) | **1형 당뇨병 (인슐린의존성)** | 이자 β세포 항원 | β세포 파괴 → 고혈당, 케토산증 | |
| | **류마티스관절염** | 관절 내 윤활막 항원 | 관절의 염증과 파괴 | 전신성 병태 |
| | **다발경화증** | 미엘린 염기성 단백질 | 신경변성, 마비 | |

> **용어**: TSH = 갑상샘자극호르몬 / snRNP = 핵 내 저분자 리보핵단백질 / scRNP = 세포질 내 저분자 리보핵단백질

#### 4) 이식면역과 종양면역

**(1) 이식면역**

- **장기이식**: 손상·기능 상실 장기를 다른 개체의 건강한 장기 또는 자신의 다른 부위와 교환하는 치료
  - 장기 제공자: **공여자(donor)** / 이식받는 자: **수여자(recipient)** / 이식 장기: **이식편(graft)**

| 용어 | 정의 |
|---|---|
| **동종항원(alloantigen)** | 인체 세포 표면의 ABO 혈액형 항원이나 HLA 항원 등 개체 간에 차이가 있는 항원. 수여자의 면역계통이 공여자의 동종항원을 '비자기'로 인식 → 면역반응 |
| **거부반응(rejection reaction)** | 이식된 장기를 제거하려는 면역반응. 장기의 생착 여부는 거부반응 정도에 달림 |
| **이식항원(transplantation antigen)** | ABO 혈액형 항원·HLA 항원 등 이식 시 문제가 되는 동종항원. 공여자-수여자 간 이식항원 최대한 일치시키는 것이 중요 |

- 공여자와 수여자의 **HLA 항원형이 완전 일치**하거나, 공여자의 HLA가 수여자의 HLA에 완전히 포함될 때 → 이식 장기 **생착**
- 조건 불만족 시 → 이식 장기 **거부**
- **이식편대숙주반응(GVHR, graft versus host reaction)**: 골수이식 시 공여자의 림프구가 숙주에 침입 → 숙주의 면역력이 저하되어 거부반응이 일어나지 않는 경우 → 공여자의 림프구가 **숙주 조직을 손상**
- **면역억제제(immunosuppressant)**: HLA 일치 공여자를 찾지 못하는 경우 수여자에게 면역억제제 투여, 방사선 조사 → 거부반응 경감 → 이식장기 생착 도모

**(2) 종양면역**

- 인체 내에서 매일 약 **5,000개의 변이세포**가 생성
- 여기에 **종양유전자(oncogene) 활성화** 또는 **종양억제유전자(tumor suppressor gene) 결핍**이 더해지면 종양 형성
- 인체의 암 방어기전:
  - DNA 복제·세포분열 상태를 감시하여 정상화하는 기전
  - **면역계통에 의한 암 발병방지기전**: 형질전환하여 **변형된 자기(altered-self, 비정상 자기)**를 인식·제거
  - 중심 세포: **NK세포** + **CD8⁺ 세포독성T세포(CTL)**

### 학습목표

2. 감염원과 감염경로를 각각 설명할 수 있다.
3. 감염의 종류를 열거하고 그 특징을 설명할 수 있다.
4. 급성 감염병과 만성 감염병의 경과 과정을 설명할 수 있다.

---

### 1. 감염의 성립

#### 1) 감염과 감염병의 정의

- **감염(infection)**: 병원체가 숙주의 생체방어기전을 뚫고 체내로 침입하거나 점막 표면에 부착하여 **증식**하는 것 → 생체에 어떠한 반응을 이끌어 내면 '감염이 성립하였다'고 함
- **감염병(infectious disease)의 발병**: 감염 후 **염증반응**으로 다음과 같은 주요 현상이 동반
  - ① 혈관 확장 및 혈류 증가 → 홍반(redness), 열감(fever)
  - ② 모세혈관벽 투과성 증가 → 체액성분의 조직 이동 → 부종(swelling)
  - ③ 호중구 포함 염증세포의 감염 부위 집합
- 감염 후 염증반응은 세균 감염 확산을 방지하고 면역반응을 일으키는 **초기방어기전**

#### 2) 감염 성립의 3요소 ★★

| 요소 | 내용 |
|---|---|
| **① 감염원 (infection source)** | 감염을 성립시키는 병원성 미생물 |
| **② 감염경로** | 미생물이 숙주(사람)에 침입하는 경로 |
| **③ 감수성 숙주** | 미생물에 감수성을 나타내는 개체 |

- 감염병 발병 여부 결정: **미생물의 병원성(독력)** vs **숙주의 저항력**
  - 병원성 > 저항력 → 감염병 발병
  - 병원성 < 저항력 → 발병하지 않음
  - 병원성 ≒ 저항력 → **무증상감염(불현성 감염, inapparent infection)**: 감염은 성립하나 발병하지 않음

---

### 2. 감염원(infection source)

- 질병에 걸린 사람이나 동물이 대표적 감염원
- 발병하지 않더라도 병원체를 배출하는 사람(보균자)이나 동물도 감염원에 해당
- **무증상감염(불현성 감염)**: 전형적 증상 없이 감염된 경우 → 감염원으로서 **위험성이 특히 높음**

| 용어 | 정의 |
|---|---|
| **보균자(carrier)** | 증상은 없지만 병원체를 보유하는 사람 |
| **무증상보균자(asymptomatic carrier)** | 과거 감염 시 증상이 경미하거나 전혀 없어 감염 사실조차 모르는 경우 |
| **회복기보균자(convalescent carrier)** | 질병에서 회복된 후 보균자가 된 경우. 혈액·분비물에서 지속적으로 감염성 미생물 배출 |

---

### 3. 감염경로

- 병원체는 다양한 경로를 통해 체내로 침입
- 전파양식에 따라 **수평감염**과 **수직감염**으로 분류

#### 1) 수평감염 — 사람에게서 사람으로 감염 확산 (4종류)

**(1) 접촉감염(contact infection)** → 표준주의 + 접촉주의

| 유형 | 기전 |
|---|---|
| **직접접촉감염** | 감염자와 직접 접촉함으로써 병원성 미생물 전파 |
| **간접접촉감염** | 오염된 감염자의 손, 물건, 환경 표면 등을 통해 전파 |

- 접촉감염 전파 미생물:
  - **항균제 내성균**: MRSA(메티실린 내성 황색포도알균), VRE(반코마이신 내성 장알균), CRE(카르바페넴 내성 장내세균과 세균), 녹농균
  - **감염성 설사 원인**: 창자출혈성 대장균, 노로바이러스, 로타바이러스, 디피실균(C. difficile)
  - 기타: RS바이러스, 단순헤르페스바이러스, 아데노바이러스(유행성 각결막염), 옴진드기

**(2) 비말감염(droplet infection)** → 비말주의

- 병원성 미생물을 포함한 **비말**이 비감염자의 얼굴 등에 부착 → 코안·입안의 점막이나 결막을 통해 침입

| 용어 | 정의 |
|---|---|
| **비말(droplet)** | 기침·재채기·대화 시 방출되는 **5μm 이상** 크기의 호흡기 유래 입자. 전파거리 비교적 짧음 (~1m) |
| **비말핵(droplet nuclei)** | 방출된 비말이 건조되어 **5μm 이하** 크기가 된 입자. 공기 중 오래 부유 → **공기전파**의 원인 |

- 비말전파 미생물: 폐렴미코플라스마, 백일해균, A군 용혈성 사슬알균(항균제요법 시작 후 24시간까지), 수막염균, 인플루엔자바이러스, 리노바이러스, 풍진바이러스, 멈프스바이러스

**(3) 공기감염(aerial infection)** → 공기주의

- 감염자에게서 방출된 비말이 짧은 시간에 **비말핵**이 된 후 **부유 비말핵(airborne droplet)**으로서 확산
- 공기의 흐름을 타고 **광범위하게 확산** → 감염자와 직접 접촉 없이도 노출 가능

| 분류 | 미생물 |
|---|---|
| 공기감염으로**만** 전파 | **결핵균** |
| 공기감염 주 경로 + 다른 경로도 가능 | **홍역바이러스**, **수두대상포진바이러스** |
| 특수 환경(한정된 공간)에서만 공기감염 | 인플루엔자바이러스 등 |

**(4) 매개체감염(vector-borne infection)** → 표준주의

- 오염된 식기, 주삿바늘, 의류 등 또는 병원성 미생물이 포함된 물, 음식물, 혈액 등의 **매개체**를 통해 감염 성립
- 수인성 식품 매개, 개달물(fomite) 매개, 혈액 매개 전파 등
- **혈액 매개 전파 미생물**: B형 간염바이러스(HBV), C형 간염바이러스(HCV), 사람면역결핍바이러스(HIV) → **의료관련감염**의 주요 원인
  - 병원성 미생물 보유자의 혈액이 묻은 주삿바늘·예리한 물건에 찔리거나 상처·점막 직접 노출 → 의료종사자 감염
  - 혈액투석실에서 HBV·HCV에 오염된 약물(헤파린 등)이나 의료기구 → 다른 환자에게 전파

**수평감염의 종류 종합 (표 3-1)**

| 종류 | 특징 | 미생물 | 질환 |
|---|---|---|---|
| **접촉감염** | 보균자·보균 동물과 직접 접촉, 성매개감염·교상감염 | 세균(임균, 매독균, 미코플라스마), 바이러스(단순헤르페스, HBV, HIV), 진균(칸디다 알비칸스), 원충(질편모충, 이질아메바, 람블편모충) | 접촉전염고름딱지증, 임질, 매독, 광견병, 파상풍 등 |
| **비말감염** | 재채기·기침 비말 흡입 | 세균(결핵균, 폐렴미코플라스마, 폐렴클라미디아), 바이러스(인플루엔자, 풍진, 멈프스) | 인플루엔자, 풍진, 유행귀밑샘염, 백일해, 세균성 폐렴, 미코플라스마폐렴 |
| **공기감염** | 5μm 이하 비말핵 흡입, 먼지감염 | 세균(결핵균, 레지오넬라균), 바이러스(인플루엔자, 홍역, 수두대상포진), 진균(아스페르길루스균) | 결핵, 홍역, 수두 등 |
| **매개체감염** | 오염된 식기·주삿바늘·의류, 물·음식물·혈액 | 세균(황색포도알균, 세레우스균, 창자출혈성 대장균, 콜레라균, 보툴리누스균, 페스트균), 바이러스(노로, 아스트로, E형 간염, 일본뇌염), 원충(말라리아원충, 파동편모충) | 식중독, B형 간염, C형 간염, AIDS 등 |

#### 2) 수직감염 — 모체 → 태아/신생아

- 모체에 감염되어 있던 병원체가 **임신, 분만, 수유** 시 태아나 신생아에게 직접 감염 확산

**수직감염의 종류 (표 3-2)**

| 종류 | 특징 | 미생물 | 질환 |
|---|---|---|---|
| **태반경유감염** | 태반을 통해 병원체가 태아 혈액으로 감염 | 세균(매독균), 바이러스(거대세포바이러스, 홍역바이러스) | 선천매독, 선천풍진증후군, 선천거대세포바이러스 감염병 |
| **산도감염** | 분만 시 산도나 모체 혈액 속 병원체에 의해 감염 | 세균(B군 사슬알균, 임균, 클라미디아), 바이러스(단순헤르페스, HIV, HBV) | 수막염, 각결막염, 클라미디아폐렴, 신생아헤르페스, AIDS, B형 간염 |
| **모유감염** | 모유를 통해 병원체 감염 | 바이러스(사람T세포림프친화바이러스, HIV) | 성인T세포백혈병, AIDS |

### 4. 병원성 미생물과 숙주의 상호관계

- 감염 성립과정에는 병원성 미생물과 숙주의 다양한 요인이 복잡하게 관여
- 이들은 감염병 발병 및 **중증도**에도 영향

| 측면 | 요인 |
|---|---|
| **병원성 미생물측** | 감염력(infectivity; 숙주에게 감염병을 일으키는 능력), 노출량, 노출부위(정착·생존·증식 가능 조직으로의 침입) |
| **숙주측** | 감수성(면역 미획득), 연령, 기저질환(영양결핍, 면역력 저하, 당뇨병, 악성종양, 화상 등), 외과수술·이식·방사선 치료, 약물(스테로이드제, 면역억제제, 항암제), 카테터·장치 등 인공물의 체내 유치 |

### Advance: 감염 회로(chain of infection) ★★★

- 감염은 일련의 과정을 통해 다양한 경로를 거쳐 전파 → 이 과정을 **감염 회로**라 함
- 효과적인 감염관리정책은 감염 회로의 **하나 또는 그 이상을 끊어내어** 감염 전달을 예방하는 것

**감염성 질환의 생성 6단계**

| 단계 | 내용 | 차단 전략 |
|---|---|---|
| **① 병원체 (causative agents)** | 바이러스, 세균, 진균, 리케차, 원충, 기생충, 프리온 등 | 소독, 멸균 |
| **② 병원소 (reservoir of infection)** | 인간(환자, 보균자), 동물, 흙, 물 등 | 위생, 상처소독, 오염물 처리, 세탁물 처리 |
| **③ 병원소로부터 병원체의 탈출 (escape from reservoir)** | 호흡기·소화기·비뇨생식기·피부 등을 통해 병원체 배출 | 상처소독, 환자의 마스크 착용 |
| **④ 전파 (mode of transmission)** | 직접전파(비말, 교상, 수직감염), 간접전파(공기, 물/식품, 곤충) | 장갑·마스크·고글 착용, 무균술식, 오염물 처리, 손위생 |
| **⑤ 신숙주에 침입 (entry into new host)** | 호흡기, 소화기, 비뇨생식기, 피부, 태반 | 멸균법 |
| **⑥ 신숙주의 감수성과 면역 (susceptibility and immunity)** | 면역(선천/후천), 영양, 건강상태 | 건강한 면역체계(예방접종, IVIG 투여), 운동, 영양, 피부보호 |

- 6가지 요소가 모두 연결되어야 감염병 전파가 이루어짐 → **어느 한 가지라도 결여되면 감염병 전파 불가** → 질병 예방전략의 핵심

> **감염성 질환 관리 방법 2가지**
> 1. **전파과정 단절**: 병원체가 숙주에게 접촉할 수 없게 철저한 환경관리 — 검역, 환자·보균자 격리(음압격리병동 등), 환경위생(소독·멸균), 식품위생, 개인위생(손씻기, 마스크)
> 2. **숙주 저항력 강화**: 면역력 강화(예방접종, IVIG 투여), 환자 조기발견 및 조기치료

### 5. 감염과 발병

#### 1) 감염병의 발병 기전

- 병원체가 숙주에 **정착·증식** → 조직손상 → 면역반응 → 감염병 발병
- 전형적 감염 증상 (염증반응): **발열(fever), 발적(redness), 종창(swelling), 통증(pain)**
- **병원체(pathogen)**: 건강한 사람에게 감염병을 일으키는 **병원성(pathogenicity)**을 가진 미생물

**인체의 병원체 침입 저지 기전**

| 부위 | 방어기전 |
|---|---|
| **피부** | 케라틴과 중층편평상피로 덮여 병원체 침입 차단 |
| **소화관** | 침 속 용균효소(lysozyme), 위산, 쓸개즙, 항균펩티드 등이 병원체 사멸 |
| **외부 접촉 부위** | 다수의 **상재균(resident)**이 병원성 미생물의 정착 저지 + 면역계통 기능 유지에 중요 |

#### 2) 건강한 사람의 감염

- 고병원성 미생물은 체내 침입 후 **병원성 인자(virulence factor)**를 생산 → 숙주 방어기전을 돌파하고 조직 내 증식

**대표적 병원성 인자**

| 병원성 인자 | 역할 |
|---|---|
| **섬모(pili/fimbriae)** | 숙주 내 정착·증식을 위한 세포 부착 |
| **편모(flagella)** | 운동성 담당 |
| **협막다당질(capsular polysaccharide)** | 면역회피 (탐식작용 저항) |
| **독소(toxin)** | 조직손상, 면역계통 교란 |

- 고병원성 미생물 예시: **백일해균(Bordetella pertussis)**, **장출혈성대장균(enterohemorrhagic E. coli)**
- **외인감염(exogenous infection)**: 고병원성 미생물은 보통 사람 체내의 상재균이 아니라 **외부에서 침입**하는 미생물
- 감염병 발병 여부: 병원성 미생물의 **양**, 면역반응의 **강도**, 숙주요인(백신접종 여부, 영양상태 등)에 좌우

#### 3) 감염의 유형 ★★

| 유형 | 정의 | 비고 |
|---|---|---|
| **무증상감염 (불현성 감염, inapparent infection)** | 병원성 미생물이 조직 내 증식하더라도 생체방어기전이 빠르게 작동하여 제거 → **증상 없음**. 면역반응은 유도되어 항체 생산 등 면역 획득 가능 | 건강한 사람에게 흔함 |
| **지속감염 (persistent infection)** | 무증상감염 후 숙주 면역반응을 **회피**하여 **지속적으로 증식** | B형 간염바이러스 감염 |
| **잠복감염 (latent infection)** | 무증상감염 후 숙주의 감염 세포 내에서 **휴면상태** → 면역계통에 인식되지 않고 장기간 존재 | 결핵균, 수두대상포진바이러스 |
| **증상감염 (현성 감염, apparent infection)** | 생체방어기전을 뚫고 미생물 증식 → 감염 조직 손상 → **심한 염증반응** → 감염 증상 발현. 치료·면역에 의해 제거되면 치유, 제거 안 되면 중증 조직손상 또는 **사이토카인 폭풍** → 사망 가능 | |

> **감염의 결과 흐름**
>
> - **불현성 감염**: 숙주가 질병을 일으키지 않음 → 임상증상 없이 회복 or 항체 형성
>   - 병원체의 발병력이 낮은 경우 / 병원체 양이 소량 / 적절한 침입로를 찾지 못한 경우 / 숙주의 면역이 작용한 경우
> - **현성 감염**: 증상과 질병이 발생 → 완전 회복, 후유증, 또는 사망
>   - 병원체 양이 다량이거나 숙주 면역이 결여될 때 / 병원체의 발병력이 높은 경우

#### 4) 면역저하숙주(immunocompromised host)의 감염

- 장기이식, 침습성 수술, 콜라겐병 치료(생물학적 제제), 당뇨병 등 대사질환, 고령화 → **면역저하숙주 증가**
- 면역저하숙주에서 감염병은 더 복잡·다양한 병리, 약제내성, 면역력 저하로 **치료 어려움**

| 감염 유형 | 정의 | 예시 |
|---|---|---|
| **기회감염 (opportunistic infection)** | 면역저하숙주에게 일어나는 감염. 저병원성 미생물(녹농균 등)에도 감염 성립. 고병원성 미생물에도 매우 적은 균량으로 감염·중증화 | 녹농균, 아시네토박터, 뉴모시스티스 등 |
| **내인감염 (endogenous infection)** | 자신이 가진 **상재균**에 의한 감염. 저영양 시 창자점막 장벽 기능 저하 → 창자 내 상재균이 혈중 침입 → **세균혈증(bacteremia)** 또는 **패혈증(sepsis)** 발병. 잠복·지속감염 미생물의 **재활성화** 용이 | 고령자 대상포진, 항암제 치료 시 B형 간염 재발, 다약제 내성 내인감염 |
| **균교대감염 (superinfection)** | 광역스펙트럼 항균제 투여 → 상재균 감소 → 그 항균제에 **저항성**을 갖는 세균 과증식. 독소 생산 세균이 과증식 시 감염병 발병 | **C. difficile의 거짓막대장염(pseudomembranous colitis)** |

> **용어 정리**
> - **세균혈증(bacteremia)**: 혈액 내 균 검출 상태
> - **패혈증(sepsis)**: 혈액 중 미생물 증식으로 쇼크, 다장기부전 등을 일으키는 위독한 상태

### Advance: 정상균무리의 공생과 불균형

- 인체의 피부·소화관에는 많은 수의 세균이 상주, 환경 유래 일과성 정착 미생물 무리도 포함
- 상재균은 세포 파괴 독소 등의 병원성 인자가 없어 감염병 원인이 되는 일은 드묾
- 상재균의 유익한 작용: 외래 병원성 미생물의 **정착·증식 억제**
- 숙주는 상재균에 대해 안정적 정착부위 제공 + **면역허용(immunologic tolerance)** 성립 → **공생(symbiosis)** 관계
- 이러한 미생물 무리 = **정상균무리(정상세균총, normal flora)**
- 지속되는 염증 → 상재균의 수와 다양성 소실 → 정상균무리의 유익한 기능 상실 → **불균형(dysbiosis)**

### 6. 감염의 경과

- 병원체의 종류, 감염 숙주의 생리상태 등 다양한 요인에 따라 다양한 경과를 나타냄

#### 1) 감염병의 경과 4단계

| 단계 | 정의 | 특징 |
|---|---|---|
| **잠복기 (incubation stage)** | 병원체에 노출된 후 **증상이 나타나기 전**까지의 기간 | 병원체에 따라 잠복기 상이. 증상 전부터 병원체 배출하는 경우 많아 **감염 확산 주의** 필요 |
| **전구기 (prodromal stage)** | **미열, 권태감** 등 전구증상이 나타나는 시기 | 병원체 증식 → 큰포식세포 등 **선천면역** 활성화 → 감염 조직에서 **염증 유도** |
| **발병기 (극기, fastigium)** | 병원체에 대한 **면역반응이 최고조**에 달하여 **염증증상이 현저**해지는 시기 | |
| **회복기 (convalescent stage)** | 탐식세포 유도, **후천면역 활성화** → 병원체 제거 → **회복·치유** | |

#### 2) 급성 감염병과 만성 감염병

**(1) 급성 감염병(acute infection)**
- 발병 → 치유까지 기간: **수일 ~ 2주일** (짧다)
- 항균제·해열제 투여가 경과에 중요한 영향
- **재발감염(recurrent infection)**: 잠복감염·지속감염을 일으키는 바이러스 등이 **수십 년 후 재활성화** → 다시 감염병 발생
  - 유발 요인: 고령화, 항암제 투여 등에 의한 면역기능 저하, 다른 바이러스 감염이 계기

**(2) 만성 감염병(chronic infection)**
- 발병 → 치유까지 기간: **수주 ~ 수년** (길다)
- 결핵, 비결핵성 항산균 감염병 등이 해당
- 면역저하숙주: 병원체 완전 제거 못한 상태에서 항균제 치료 중단 → 병원체 재증식 → 증상 재발
- **균막(biofilm)**: 체내 삽입된 의료용 장치(혈관 유치 카테터, 인공심장판막, 인공관절 등) 표면에 세균이 부착 → 균체 밖으로 다당질 분비 → 부착 표면막 형성
  - 균막 내부 균: **탐식세포 작용 회피** + **항균제 침투 어려움**
  - 균막 생성 의료용 장비 → **재발 반복하는 만성 감염병**의 원인

#### 3) 보균자의 감염 경과

- **무증상보균자**: 병원체가 체내에 지속 감염되어 있으나 뚜렷한 증상 없음 → **감염원으로서 중요**
- 예: B형 간염에 걸린 모체에게서 태어난 신생아 → 출생 후 백신접종·면역글로불린 투여 등 적절한 조치 없으면 **HBV 보균자**가 됨
- HBV 보균자: 후천면역이 HBV에 대한 특이적 면역반응을 일으키기까지 무증상이지만 **혈중에 감염성 HBV 존재**

### Focus: 홍역(Measles) — 제2급 법정감염병 사례 학습

| 항목 | 내용 |
|---|---|
| **병원체** | Paramyxovirus군 Measles virus |
| **잠복기** | 7~21일 (평균 10~12일) |
| **전염기** | 발진 **4일 전 ~ 4일 후** (D-4일~D+4일) |
| **증상** | 발열, 결막염, 감기, 기침, **구강점막 코플릭 반점(Koplik's spot)** 등 전구증상 → 3~7일째 얼굴부터 붉은 발진 → 4~7일 후 탈락 |
| **병원소** | 사람 |
| **전파양식** | 감염자 코/목 분비물 직접접촉, **비말 또는 공기전파** |
| **전염성** | 매우 높음 (**R₀ = 12~18**) |
| **발생 양상** | 겨울, 봄에 호발 |

> **코플릭 반점(Koplik's spot)**: 첫 번째 하구치 맞은편 구강점막에 충혈된 작은 점막으로 둘러싸여 있는 회백색의 모래알 크기 작은 반점

**홍역의 예방관리 (1·2·3차 예방)**

| 구분 | 방법 |
|---|---|
| **1차 예방** | 교육(환자 접촉 회피), 면역(**생후 12~15개월, 4~6세 MMR 접종**; 유행 시 생후 6개월부터 접종 가능), 감염 후 **영구면역** 형성 |
| **2차 예방** | 접촉자 항원검사, 노출 **72시간 이내 MMR 접종**으로 방어 가능 |
| **3차 예방** | 환자: 발진 전 6~7일 ~ 발진 후 2~3일까지 높은 전염력 → **발진 이후 적어도 5일간 격리 치료** |

**홍역 환자 발생 시 유행 대응**
- ① 접촉자의 **신속한 확인** (보수적으로 범위 설정)
- ② 추가 감염 예방조치: **MMR 접종 또는 IVIG 투여**
- 비말·공기전파 → 동일 장소에 머물렀던 자 **모두 접촉자**
- 전염기간(D-4~D+4)이므로 **발진일 정확한 확인** 중요 → 신속 격리로 추가 노출 방지
- 잠복기 7~21일 → 발진 발생일 기준 **3주 전~1주 사이 동선**에서 만난 누군가가 선행 환자

> **한국의 홍역 관리 현황**
> - 2000~2001년 약 5만 명 대유행 이후 홍역퇴치 사업 성공적 수행
> - 2006년 서태평양국가 최초 홍역퇴치 선언
> - 2014년 3월 WHO로부터 **국가홍역퇴치인증** 획득
> - WHO 홍역퇴치국가 인증기준: '토착화된 홍역 바이러스'에 의한 환자가 **36개월 이상** 발생하지 않은 경우

### 1. 세균의 분류

#### 1) 세균(bacteria)의 특징

- 약 **1μm** 크기의 **자가복제능력**을 가진 생물
- 유전체 DNA는 **핵막으로 덮여 있지 않고** 세포질에 직접 노출 → **원핵생물(prokaryote)**로 분류
- 보통 **인공배지에서 배양 가능**
- 예외: **클라미디아(Chlamydia)**, **리케차(Rickettsia)** — 유전자 결손으로 진핵생물의 세포질에 기생하여 대사계통을 이용해야 증식 가능 (인공배지 배양 불가)

#### 2) 명명법

- 세균의 학명(scientific name): **속명(genus) + 종명(species)** 조합하는 **이명법**
- 국제적으로 학명은 **이탤릭체**, 속명의 첫 글자는 **대문자**, 종명은 모두 **소문자**로 표기

| 한국어명 | 속명 | 종명 | 약칭 |
|---|---|---|---|
| 대장균 | *Escherichia* | *coli* | *E. coli* |
| 황색포도알균 | *Staphylococcus* | *aureus* | *S. aureus* |
| 콜레라균 | *Vibrio* | *cholerae* | *V. cholerae* |

#### 3) 계층적 분류체계

- **종(species) → 속(genus) → 과(family) → 군(part) → 문(division)**
- 분류학적 등록 균종: 약 **12,000종**, 인간에 질병 유발 균종: 약 **2,000종** (기회감염균 포함)
- 분류 기준: 세균의 형태(알균/막대균/나선균), 그람염색성(양성/음성), 산소요구성(산소성/무산소성), 아포 형성 여부
- 최근에는 **DNA 염기서열(16S rRNA 배열)**을 이용한 분자생물학적 동정방법도 활용

**세균 분류 체계도 (주요 균종)**

| 형태 | 산소요구성 | 그람염색 | 대표 균속 |
|---|---|---|---|
| **알균(구균, coccus)** | 산소성 | 그람양성(+) | 포도알균속(*Staphylococcus*), 사슬알균속(*Streptococcus*), 창자알균속(*Enterococcus*) |
| | 산소성 | 그람음성(−) | 나이세리아속(*Neisseria*): 임균(*N. gonorrhoeae*), 수막염균(*N. meningitidis*) |
| | 무산소성 | 그람양성(+) | 펩토사슬알균속(*Peptostreptococcus*) |
| | 무산소성 | 그람음성(−) | 베이요넬라속(*Veillonella*) |
| **막대균(간균, bacillus)** | 산소성 | 그람양성(+) | 바실루스속(*Bacillus*), 리스테리아속(*Listeria*), 코리네박테륨속(*Corynebacterium*), 락토바실루스속(*Lactobacillus*) |
| | 산소성 | 그람음성(−) | 장내세균과(*Enterobacteriaceae*), 비브리오속(*Vibrio*), 헤모필루스속(*Haemophilus*), 슈도모나스속(*Pseudomonas*), 보르데텔라속(*Bordetella*) |
| | 산소성 | 항산성(+) | 미코박테륨속(*Mycobacterium*) |
| | 무산소성 | 그람양성(+) | 클로스트리듐속(*Clostridium*), 프로피오니박테륨속(*Propionibacterium*) |
| | 무산소성 | 그람음성(−) | 박테로이데스속(*Bacteroides*), 프레보텔라속(*Prevotella*), 푸소박테륨속(*Fusobacterium*) |
| **기타** | — | — | 미코플라스마(*Mycoplasma*), 스피로헤타(*Spirochaetes*): 트레포네마속, 보렐리아속, 렙토스피라속 |
| **인공배지 미배양** | — | — | 클라미디아(*Chlamydia*), 리케차(*Rickettsia*) |

### 2. 세균의 형태와 구조

#### 1) 크기와 형태 — 3가지 기본형 ★★

#### 2) 구조와 기능

- 세균 = **단세포 원핵생물**, 매우 단순한 구조
- 핵막, 핵소체, 미토콘드리아, 세포질그물(소포체) 등 **없음**

**세균의 주요 구조물과 기능**

| 구조물 | 기능 |
|---|---|
| **세포막(cell membrane)** | 세포질을 둘러싸는 막 |
| **세포벽(cell wall)** | 세포막 바깥쪽의 견고한 층 → 세균 **형태를 일정하게 유지** |
| **협막(capsule)** | 세포벽 바깥 점조성 층 → **식균작용에 대한 저항성** 높여 **병원성 강화** |
| **편모(flagellum)** | **운동기관** |
| **섬모(cilia/pili)** | **부착**, 접합 시 기능 |
| **핵양체(nucleoid)** | 핵영역, 섬유 모양 DNA |
| **플라스미드(plasmid)** | 염색체 바깥 유전자(고리 모양 DNA) → 약물내성, 병원성 인자 유전자 존재, 다른 균종으로 **전파** 가능 |
| **리보솜(ribosome)** | 단백질 합성 |
| **저장과립(storage granule)** | 대사산물 저장 (디프테리아균의 이염소체 → 균 동정 중요 소견) |

| **테이코산** | **+** | − |
| **구멍단백질(porin)** | − | **+** (물질흡수 관여) |

**그람양성균 세포벽 특징**
- **두꺼운 펩티도글리칸** 층
- **테이코산(teichoic acid)**, **리포테이코산(lipoteichoic acid)** 존재

**그람음성균 세포벽 특징**
- **얇은 펩티도글리칸** 층 → 바깥막(외막)과 속막(내막) 사이에 존재
- **지질다당질(LPS, lipopolysaccharide)**: **지질 A** + 다당류 + **O항원** → **내독소(endotoxin)**로 활성
- **외막단백질 구멍단백질(porin)**: 삼량체 형성, 단량체는 작은 구멍으로 물질흡수 관여

#### 4) 세포질(cytoplasm) 상세

| 구성요소 | 특징 |
|---|---|
| **핵양체(nucleoid)** | 섬유 모양 DNA 영역, 핵영역 (핵막 없음) |
| **플라스미드(plasmid) DNA** | 고리 모양, 염색체 바깥 유전자. 약물내성·병원성 인자 유전자 → 다른 균종으로 전파되는 경우 많음 |
| **리보솜(ribosome)** | 세포질 내 단백질 합성 소기관, 다수 존재 |
| **저장과립** | 대사산물 저장. 디프테리아균 **이염소체(metachromatic granule)** → 균 동정 중요 소견 |

#### 5) 협막(capsule) ★★

- 세포벽 바깥쪽 **점조성 점액층(slime layer)**
- **글리코칼릭스(glycocalyx)**: 세균 세포 표면 분비되는 점액성 물질, 주로 **다당류**로 구성 (탄저균은 D-글루탐산 폴리펩티드)
- 협막의 존재 = 세균의 **병원성과 깊은 관계** → 숙주의 면역반응 회피
  - **식균작용에 높은 저항성**
  - **보체에 의한 용균작용**으로부터 균체 보호
- **협막 형성균**: 탄저균, 폐렴사슬알균, 인플루엔자균, 수막알균, 폐렴막대균, 백일해균

#### 6) 편모(flagellum) ★★

- **운동성**이 있는 세균의 운동기관
- 지름 12~30nm, 길이 1~70μm
- **플라젤린(flagellin)** 단백질로 구성 → 항원성을 가짐 (**H항원**)
- 편모의 수와 부착 위치는 균종에 따라 다름

| 분류 | 구조 | 대표 균종 |
|---|---|---|
| **무모균** | 편모 없음 | 이질균, 폐렴막대균 |
| **단모균(극단모균)** | 한쪽 끝에 1개 | 콜레라균, 장염비브리오균, 에로모나스, 녹농균, 레지오넬라 |
| **양모균** | 양쪽 끝에 1개씩 | 캄필로박터 |
| **총모균(극다모균)** | 한쪽 끝에 여러 개 | 스피릴룸(나선균), 쁠레시오모나스, 세라티아 |
| **주모균** | 전체 주위에 수많이 | 대부분 장내세균, 클로스트리듐속, 바실루스속, 리스테리아* |

> *리스테리아는 25℃ 전후에서만 보통 5개 이하의 적은 편모를 형성

#### 7) 섬모(cilia/pili)

- 편모보다 **짧은** 구조물, 특히 **그람음성균**에서 많이 관찰
- 균체 주위에 바깥을 향해 섬유상으로 존재
- **운동성과는 관계없음**

| 종류 | 기능 |
|---|---|
| **성섬모(sex pilus) / 접합섬모(conjugative pilus)** | 세균-세균 부착 기능. 다른 세균과 결합하여 **전달성 플라스미드**를 상대 세균으로 전달 |
| **부착섬모** | 숙주 세포 표면에 **부착** → 그곳에서 증식 → **세균의 감염 성립**에 중요한 역할 |

#### 8) 아포(spore) ★★★

- 나쁜 환경(영양부족, 건조 등)에서 **저항성을 갖는 아포** 형성 → **자기보존**, 좋은 환경에서 재증식에 대비
- 세균의 **휴면형** (대사 멈춤, 생명 유지) → 좋은 환경에서 **발아(germination)** → 영양 균체로 이분열 증식

**아포의 특징**
- **100℃ 가열에도 장시간 견딤** → 멸균에 **고압증기멸균** 필요
- 에탄올 등 많은 소독제 및 화학적 처리에도 **저항성 강함**
- **건조에도 강함**

**아포 형성 대표 균종**

| 속 | 균종 | 아포 위치 |
|---|---|---|
| **바실루스속(Bacillus)** | 탄저균(*B. anthracis*), 세레우스균(*B. cereus*), 고초균(*B. subtilis*) | 중심성 (탄저균) |
| **클로스트리듐속(Clostridium)** | 파상풍균(*C. tetani*), 보툴리누스균(*C. botulinum*), 웰치균(*C. perfringens*) | 단재성(파상풍균), 편재성(보툴리누스균) |

### 3. 관찰방법

#### 1) 광학현미경 — 염색법

**(1) 그람염색법(Gram staining)** ★★★

- 한스 크리스티안 그람(덴마크 의사)이 개발
- 세균의 구조·구성성분 특징을 활용한 염색법 → 세균 동정의 가장 기본적인 방법

**그람염색 순서**

| 단계 | 시약 | 결과 |
|---|---|---|
| ① 도말, 건조, 고정 | — | — |
| ② 1차 염색 | **크리스탈바이올렛(crystal violet)** | 모든 세균이 **보라색** 염색 |
| ③ 매염 | **루골(Lugol) 용액 (Iodine)** | 염색을 견고히 |
| ④ 탈색 | **95% 알코올(Ethanol)** | 그람음성균만 **흰색 탈색** |
| ⑤ 대조염색 | **사프라닌(safranin)** | 그람음성균만 **분홍색(적색)** 염색 |

- **그람양성균** → **보라색** / **그람음성균** → **분홍색(핑크색)**

**대표적인 병원성 세균의 그람염색 분류**

| 형태 | 그람양성 (보라색) | 그람음성 (핑크색) |
|---|---|---|
| **막대균** (유아포균) | 파상풍균, 가스괴저균균, 보툴리누스균, 세레우스균 | — |
| **막대균** (무아포균) | 결핵균*, 나균*, 디프테리아균, 리스테리아, 악티노미세스, 노카르디아 | 대장균, 폐렴막대균, 장염비브리오균, 콜레라균, 이질균, 녹농균, 유비저균, 레지오넬라, 아시네토박터*, 모락셀라*, 브루셀라*, 캄필로박터*, 헬리코박터*, 백일해균*, 박테로이데스 |
| **나선균** (스피로헤타) | — | 트레포네마*, 렙토스피라*, 보렐리아* |
| **알균** | 황색포도알균, 사슬알균, 장알균균 | 임균, 수막염균, 베이요넬라 |

> *1: 균체는 구부러져 S자 나선 모양 / *2: 짧은막대균(일부 알균과 유사) / *3: 그람염색에 잘 염색되지 않음

**(2) 항산염색법(acid fast staining)**

- 대표 항산균: **결핵균(*Mycobacterium tuberculosis*)**
- 잘 염색되지 않지만, 일단 염색되면 **산이나 알코올에 잘 탈색되지 않는 성질** 이용
- **칠넬센염색법(Ziehl-Neelsen staining)**이 대표적

| 단계 | 시약 | 결과 |
|---|---|---|
| ① 가온 염색 | **카르볼푹신(carbolfuchsin)** | 모든 세균 붉게 염색 |
| ② 탈색 | **3% 염산알코올** | 항산균은 탈색 안 됨, 기타 세균은 탈색 |
| ③ 대비염색 | **메틸렌블루** | 기타 세균 파랗게 염색 |

- 항산균 → **붉은색** / 기타 세균 → **푸른색**

**(3) 기타 염색법**

- **단염색법(simple staining)**: 슬라이드글라스에 도말한 세균을 수분 동안 단일 염색시약으로 염색 관찰하는 방법
- **세균의 특징적 구조를 관찰하는 특수 염색법**:

| 염색법 | 영문 | 대상 구조 | 대표 균종 |
|---|---|---|---|
| **편모염색법** | flagellar staining | 편모 | 장티푸스균(*Salmonella typhi*) 등 |
| **이염소체염색법** | metachromatic body staining | 이염소체(저장과립) | 디프테리아균(*Corynebacterium diphtheriae*) |
| **협막염색법** | capsule staining | 협막 | 탄저균(*Bacillus anthracis*) 등 |
| **아포염색법** | spore staining | 아포 | 바실루스속, 클로스트리듐속 |

> **참고**: 탄저균의 협막 염색 — 마우스에 접종 후 사망시킨 뒤 비장(지라)을 메틸렌블루로 염색하면, 균체 주위의 옅은 염색 부분이 협막에 해당

#### 2) 전자현미경에 의한 관찰

- 세균의 미세 구조를 관찰하는 데는 **전자현미경**이 필요

| 종류 | 원리 | 관찰 대상 |
|---|---|---|
| **주사전자현미경(SEM)** | 전자선으로 시료 **표면**을 스캔 → 방출되는 이차전자·반사전자를 검출하여 영상화 | 균 전체의 **형태**, 균체 표면의 성질, **배열**, 상세한 분열 상태 |
| **투과전자현미경(TEM)** | 초박절편 시료에 전자선을 쏘아 **투과**한 전자가 만드는 상을 확대 관찰 | 분해능력이 높아 균체 내의 **초미세 구조** 관찰에 유용 |

> 기억 포인트: **SEM = 형태(표면)** / **TEM = 초미세구조(내부)**

### 3. 세균의 증식

- 세균은 **단세포생물**이며, 균수가 증가하는 것을 **증식(multiplication)**이라고 함
- 세균에 따라서는 환경에 대응하여 크기나 형태를 다소 바꿀 수 있음
- 증식은 하나의 세균세포가 2개로 분열하는 **이분열(binary fission)**의 양식으로 일어남
  - 과정: DNA 복제 → 복제된 DNA가 좌우로 나뉘고 세포벽 형성 → 2개의 세포로 분열 → 2개의 딸세포 형성

#### 1) 영양(nutrition)

- 세균이 증식하려면 **영양을 섭취**해야 함

| 분류 | 영문 | 특징 |
|---|---|---|
| **독립영양균(자력영양균)** | autotrophic bacterium | **무기물만**으로 증식 가능 |
| **종속영양균(타력영양균)** | heterotrophic bacterium | 무기물 외 식물 등이 생산한 **유기물** 필요 |

- **병원성 세균**은 **종속영양균**에 속함
- 필요 영양소: 탄소원, 질소원, 인산, 칼륨, 나트륨, 마그네슘, 무기염류(칼슘, 철, 망간, 코발트 등)

| 영양소 | 세부 내용 |
|---|---|
| **탄소원** | 식물이 광합성한 포도당을 비롯 각종 탄수화물(단당, 올리고당, 다당)을 탄소·에너지원으로 이용 |
| **질소원** | 아미노산 합성에 반드시 필요. 일부 세균은 아미노산 합성 불가하여 탄소도 아미노산에 의존 |
| **미량의 무기염류** | 핵산이나 아미노산의 구성원소(인, 황, 마그네슘 등) 외의 미량 무기염류는 배지성분이나 유리 기구에서 용출되므로 별도 첨가 불필요 |
| **발육인자** | 미량이면서 증식에 반드시 필요하지만 **스스로 합성할 수 없는 성분** — 비타민, 핵산의 염기(퓨린, 피리미딘) 등 |

> **위성현상(satellitism)**: 황색포도알균이 섞이면 그 주변에서 인플루엔자균이 증식하는 현상 → 인플루엔자균에 필요한 **X인자(헤민)**와 **V인자(NAD)**를 황색포도알균이 공급하기 때문

#### 2) 증식에 관여하는 인자

**(1) 수분**
- 세균은 건조에는 비교적 강하지만, 증식을 하려면 **수분이 반드시 필요**

**(2) 온도** ★★
- 세균에는 증식에 가장 적합한 **최적온도(optimum temperature)**가 있음

| 분류 | 온도 범위 | 비고 |
|---|---|---|
| 저온균 | −5~30℃ | |
| **중온균** | **25~45℃** | **병원성 세균 대부분** — 사람 체온에 가까운 **37℃**가 최적 |
| 고온균 | 40~70℃ | |
| 고도호열균 | 70℃ 이상 | |

**(3) 수소이온농도(pH)**
- 세균에 따라 증식에 가장 적합한 **최적 pH(optimal pH)**가 있음
- 대부분 **pH 5.0(약산성)~pH 8.0(약알칼리성)** 범위에서 생존 및 증식, **pH 7.0(중성)** 중심
- **헬리코박터 파일로리(*Helicobacter pylori*)**: 위산 분비 산성환경에서 증식, **위궤양** 유발
- **유산균(*Lactobacillus*)**: 최적 pH 5.5~6.2

**(4) 삼투압**
- 대부분의 미생물은 높은 염분농도에서 **발육 억제**
- **호염균(halophilic bacterium)**: 3~9% 또는 그 이상의 최적 염분농도를 필요로 하는 균
- 세균은 **펩티도글리칸** 구성 세포벽을 가져 외부 삼투압 변화(고장 또는 저장)에 견딜 수 있음

**(5) 산소와 이산화탄소** ★★★

| 분류 | 영문 | 산소 요구성 | 대표 균종 |
|---|---|---|---|
| **절대산소성균(편성 호기성균)** | obligate aerobe | 증식에 산소를 **반드시 필요**로 함. 산소를 사용하는 **호흡**으로 에너지 획득 | 항산균, 백일해균, 레지오넬라, 녹농균 |
| **미세산소균(미호기성균)** | microaerophile | **3~10% 정도의 낮은** 산소농도를 최적으로 함 | 캄필로박터, 헬리코박터 |
| **조건무산소성균(통성 혐기성균)** | facultative anaerobe | 산소가 **있든 없든** 증식함. 산소 있을 때 호흡, 없을 때 발효. 단, 산소 있을 때 증식이 활발 | 포도알균, 대장균, 이질균 등 |
| **절대무산소성균(편성 혐기성균)** | obligate anaerobe | 산소가 있으면 **증식 불가**, 산소가 있으면 **사멸** | 파상풍균, 보툴리누스균 등 |

- 세균의 증식에 **이산화탄소**도 필요하지만 세균 자신이 대사를 통해 이산화탄소를 생산하므로 대부분 충분
- **5~10%의 고농도 CO₂**를 필요로 하는 세균: **캄필로박터, 임균** 등 → 배양 시 이산화탄소 농도를 높여야 함

#### 3) 배양(culture)

- 증식에 필요한 영양과 환경조건을 만들어 세균을 **인공증식**하는 것
- **병원체 동정**과 **감염병 진단**에 반드시 필요한 기술

**배양의 5단계 기본 방법**

| 단계 | 영문 | 내용 |
|---|---|---|
| **접종** | inoculation | 시료(inoculum)를 미생물이 증식할 수 있는 배지에 심는 것 |
| **배양** | incubation | 배지 접종 후, 시료 속 미생물 개체를 성장 및 번식하게 하는 과정 |
| **분리** | isolation | 콜로니를 분리하여 현미경으로 관찰 → 단 하나의 미생물 종만 관찰 |
| **검사** | inspection | 시료를 현미경 관찰, 염색 등을 통해 미생물 특성 조사 |
| **동정** | identification | 대상 미생물 종을 판별, 어떠한 특성이 있고 어디에 속하는지 알아내는 것 |

**(1) 배지(culture media)**: 세균을 배양하는 장소

| 종류 | 영문 | 특징 |
|---|---|---|
| **액체배지** | liquid medium (broth) | 세균 증식을 배지의 **혼탁 정도**로 판단 (투명한 것이 기본). 삼각플라스크, 시험관, 병 등 사용 |
| **고형배지** | solid medium | 액체배지에 **한천(agar)** 첨가. 목적에 따라 **평판배지, 고층배지, 사면배지, 반고층배지** 등 사용 |

**(2) 배양법**

- 세균의 배양에는 **부란기(항온기, incubator)**가 사용

| 배양법 | 대상균 | 방법 |
|---|---|---|
| **유산소배양(호기배양)** | 산소성균(호기성균), 조건무산소성균(통성 혐기성균) | 고형배지를 그대로 부란기에 보온·배양. 액체배지는 **진탕**하면 산소 접촉면적이 넓어져 산소성균의 증식효율이 좋아짐 |
| **무산소배양(혐기배양)** | 무산소성균(혐기성균), 미세산소균(미호기성균) | 세균 접종 배지를 **무산소 상태**에서 배양. 플라스틱 백·밀폐 용기 등으로 내부를 밀폐, 화학적으로 무산소 상태를 유지하여 보온·배양 |

#### 4) 증식곡선(growth curve) ★★

- 일정량의 한정된 **액체배지**에 소량의 세균을 접종·배양 시 **증식곡선**을 얻을 수 있음

| 시기 | 영문 | 특징 |
|---|---|---|
| **① 유도기** | lag phase | 일정기간 균수가 **늘지 않는** 시기, 증식을 **준비**하는 시기 |
| **② 대수증식기** | log phase | 세균이 일정 속도로 **이분열을 반복**하여 **활발하게 증식**하는 시기. 아포형성능력 있는 세균은 **아포 형성 시작** |
| **③ 정지기** | stationary phase | 증식을 **멈추는** 시기, 균수는 늘지 않음. 유용한 **이차대사 산물**을 생산함 |
| **④ 쇠퇴기** | decline phase | 생균수가 일정한 속도로 **감소**하는 시기, 완만한 기울기로 천천히 감소 |

> **이차대사산물**: 생명체에 있어 일반적 성장·발달·생식에 직접 관여하지 않는 유기 화합물

**세대시간(generation time)**
- 균수가 **2배가 되는 데 필요한 시간**
- 장염비브리오균 등: 세대시간 약 **20분** (짧음)
- 결핵균: 세대시간 약 **18시간** (김)
- **도달 균수 = 처음 세균 수 × 2ⁿ** (n = 세대수)
- 예) 세대시간 20분인 식중독균 10개 → 60분 후 80개 → 4시간 후 **40,960개** (10 × 2¹²)
- 이러한 근거에서 **세균에 의한 식품 오염은 공중 위생상 주의**가 필요

### 4. 세균의 대사

- 세균의 배양과 증식은 **대사(metabolism)**에 의해 가능
- **분해대사(이화, catabolism)**: 포도당 등의 유기화합물을 분해하여 **에너지**를 얻는 과정
- **합성대사(동화, anabolism)**: 영양소를 재료로 하여 **세포구조를 합성**하는 과정

#### 1) 분해대사(이화)

- 포도당을 분해대사하여 생성된 에너지는 **화합물(ATP)**이나 **양성자(proton) 농도기울기**로 변환되어 물질 합성에 사용

**(1) 기질수준 인산화(substrate-level phosphorylation) = 발효**

- 전자전달계를 **통하지 않는** 포도당의 분해대사
- 인산기 전이를 촉매하는 효소반응을 통해, 효소의 **기질(substrate)이 갖고 있는 인산기를 ADP에 직접 전달** → ATP 합성
- **해당작용** 내 ATP 합성 과정, **TCA 회로** 내 GTP(결과적으로 ATP) 합성 과정에서 발생
- 산소가 없는 조건에서 이루어지는 반응이기 때문에 **발효(fermentation)**라고도 불림

**(2) 산화적 인산화(oxidative phosphorylation) = 호흡** ★★

- TCA 회로에서 형성된 **환원형 NAD, FAD**가 막 **전자전달계**에서 환원 → 최종적으로 **산소가 환원되어 물(H₂O)**이 됨 = **호흡(respiration)**
- 전자전달과정에서 **H⁺(양성자)**가 막(세균 세포막) 바깥쪽으로 배출 → **농도기울기 형성**
- 양성자가 막 안쪽으로 들어오는 에너지를 사용해 **ADP + Pi(무기인산) → ATP 합성**
- 산소 외 **질소, 푸마르산** 등이 최종 전자수용체가 되는 경우 → **무산소호흡(혐기성 호흡, anaerobic respiration)**

| 비교 | 기질수준 인산화 (발효) | 산화적 인산화 (호흡) |
|---|---|---|
| 산소 조건 | **무산소 조건** | **산소 조건** |
| 경로 | 해당작용 | 해당작용 → TCA 회로 → 전자전달계 |
| ATP 생산량 | **2 ATP** | **38 ATP** |
| 최종 산물 | 젖산(2), 또는 에탄올(2)과 CO₂(2) | CO₂(6)와 H₂O(6) |

#### 2) 합성대사(동화)

- 모든 영양소를 재료로 **단백질, 핵산, 다당, 지질** 외에 더 큰 고분자물질을 합성하고 이것을 **막, 세포벽, 염색체** 등의 구조로 만들어가는 과정
- **펩티도글리칸(peptidoglycan)**: 세포벽 구성 성분으로, 세균에만 존재하는 구조물 → 생합성 경로도 독특
- **그람음성균의 바깥막(외막)**: 역시 세균에만 존재
- 이 구조물의 합성을 억제하는 물질은 **선택독성이 높은 항균제**가 될 수 있음

### 5. 세균의 유전

- **유전(inheritance)**: 모세포에서 생성된 딸세포가 그 개체의 특성을 이어받아 다음 세대로 전달하는 현상 (모든 생물에서 동일하게 일어남)
- 개체의 특성은 **DNA(데옥시리보핵산, deoxyribonucleic acid)**에 유전정보로 조합되어 있음
- 각각의 유전정보를 담당하는 단위 = **유전자(gene)**
- 유전자를 통합한 것 = **유전체(게놈, genome)**
- 세균의 유전체는 **염색체(chromosome)**와 **플라스미드(plasmid)**의 2가지 형태로 존재

#### 1) 유전체

**(1) 염색체(chromosome)**
- **이중가닥(double-stranded, ds) DNA**로, 핵막으로 싸여 있지 않고 **세포질 내에 존재**
- 염색체의 **복제개시점**에서 양쪽을 향해 **DNA 복제(replication)**가 이루어짐
- 이중가닥의 일부분이 풀어져 단일가닥이 되고, 각각의 단일가닥이 주형이 되어 **반보존적**으로 DNA 가닥을 합성 → 최종적으로 원래 이중가닥 **고리형 DNA**가 2개 생성

**(2) 플라스미드(plasmid)**
- 염색체 외에 **스스로 복제할 수 있는** 작은 유전물질
- 세균의 성장에 **필수 요소는 아니지만**, **약물내성**이나 **병원성** 등 세균의 성질과 관련하여 중요한 유전정보를 포함
- **접합, 형질전환, 형질도입** 등의 **수평적 유전자전달**을 통해 종종 다른 세균으로 이동

#### 2) 변이(variation) ★★

- 세균의 형질(형태나 생리적·생화학적 성질) 변화에는 **비유전적인 것**과 **유전적인 것**이 있음

| 분류 | 설명 |
|---|---|
| **환경변이(environment variation)** | 비유전적 변화. 동일한 유전자 구성을 가진 집단 속 개체가 성장 시 **환경 영향**으로 생기는 변화. 환경 복구 시 이전과 동등한 집단이 됨 |
| **유전변이(genetic variation)** | 유전적으로 형질에 변화가 생기는 것 |

**유전변이의 유형**

| 변이 유형 | 예시 |
|---|---|
| **형태 변화** | 편모·섬모·협막의 소실, 항원성 변화, 아포 형성능력 소실 등 |
| **한천배지 상 집락형성 변화** | 매끄러운 집락(S형) → 거친 집락(R형) 변화, 편모형성균이 유주하지 않고 균 덩어리 형성, 점액성 집락 형성능 소실 등 |
| **생화학적 성질 변화** | 당 분해능력, 색소·인돌·H₂S 생산능력 변화 등 |
| **저항성(내성) 변화** | 항생물질, 소독제, 파지에 대한 내성 변화 등 |
| **독력 변화** | 독소생산주 → 약독주 또는 무독주로 변화 |

#### 3) 유전자의 전달 ★★★

- **수평적 유전자전달(horizontal gene transfer)**: DNA(유전자)가 세균 간에 이동하는 것으로, **접합, 형질도입, 형질전환** 등이 있음
- 세균이 외래성으로 유전자를 획득함으로써 **다양성을 확보하고 진화**하는 데 중요한 역할
- **항원인자의 획득**이나 세균의 **약제내성**에도 밀접하게 관여

**(1) 접합(conjugation)**

- 균과 균의 **물리적 접합**을 통해 DNA가 이동하는 기전
- 예) 대장균의 **F인자(F플라스미드)**, 약제내성 플라스미드 등 **전달성 플라스미드**에 의한 유전자전달
- **전달성 플라스미드를 가진 균(공여균)**과 갖지 않는 균(수용균)을 공존 시 물리적 접합 → 플라스미드 DNA의 **단일가닥만** 공여균에서 수용균으로 이동
- 그람음성 막대균(대장균, 이질균 등)은 세균끼리 **직접접합** → 유전자 DNA 이행, 유전자 재조합

> **대장균 F플라스미드의 예**: F⁻균 ⇔ F⁺균 ⇔ Hfr균 ⇔ F'균
> - F⁺균(공여균)이 **성섬모**를 통해 F⁻균(수용균)에 플라스미드를 전달 → 수용균도 F⁺균이 됨
> - F플라스미드가 염색체에 삽입된 상태 = **Hfr균** → 접합 시 플라스미드 전달개시점을 선두로 **염색체**까지 전달
> - Hfr균에서 플라스미드가 부정확하게 절단되어 염색체 DNA 일부를 포함한 플라스미드 = **F'플라스미드** → F'균

**(2) 형질도입(transduction)**

- 세균에 감염되어 그 세포 내에서만 증식하는 바이러스인 **박테리오파지(bacteriophage)**를 통한 유전자 이동 기전
- 박테리오파지의 생활사 차이에 따라 2종류가 있음

| 종류 | 매개 파지 | 특징 |
|---|---|---|
| **보편형질도입** | **독성파지(virulent phage)** | **모든 유전형질**이 운반됨. 파지입자가 형성될 때 잘못해서 **숙주세균 유래의 DNA를 삽입**한 자손파지가 만들어져 다른 균에 재감염 |
| **특수형질도입** | **약독파지(temperate phage)** | **특정 유전형질만** 운반됨. 용원화한 파지가 자극에 의해 용균이 유발될 때 DNA의 **잘못된 절단**이 이루어지는 것에 따름 |

**박테리오파지의 생활사**

| 주기 | 과정 |
|---|---|
| **용균주기** | 독성파지의 감염 → 파지 DNA와 단백질 합성 → 파지입자 조립 → 파지 DNA가 파지입자 머리에 삽입 → 세균의 **용해(용균)**와 증식한 파지입자 방출 |
| **용원화주기** | 약독파지의 감염 → 숙주 유전체 DNA의 특정 부위에 파지 유전체 DNA가 삽입(**용원화, 잠복**) → 숙주 유전체와 함께 용원화 파지(**프로파지**) DNA가 복제 → 숙주의 이분열 증식과 동시에 프로파지의 DNA 복제 |

> **용원화 파지 유도**: 자외선이나 화학물질(마이토마이신 C, 아크리딘오렌지) 등 외적 자극에 의해 용원화 파지가 유도되어 **용균주기**로 전환될 수 있음

> **파지전환(phage conversion)**: 약독파지에 세균독소 등의 유전자가 부호화되어 있으며, 파지 감염 및 용원화에 의해 **독소생산능이나 새로운 형질을 균이 획득**하는 것

**(3) 형질전환(transformation)** ★★

- 세균이 **환경으로부터 특정 DNA 단편을 균체 내로 직접 삽입**하여 새로운 유전형질이 발현되도록 하는 기전 (유전형질이 바뀌는 것)
- 1928년 **그리피스(Griffith)**: **폐렴사슬알균**을 이용한 마우스의 감염실험에서 발견
  - 협막 비생산 무독주(R형균, 생균) + 협막 생산 강독주(S형균, 사균)의 혼합액을 마우스에 접종 → **마우스 치사** → 균의 분리배양 시 S형 생균과 R형 생균 모두 발견
  - R형 생균이 S형 사균이 가진 **협막 생산 형질을 획득**(형질전환능력)
- **에이버리(Avery)** 등: S형 사균에서 유리된 유전물질(**협막유전자를 부호화하는 DNA**)이 R형 생균에 삽입되었기 때문임을 밝힘 → **형질전환 유발 유전물질이 DNA에 있음**을 생화학적으로 증명
- 자연형질전환 능력을 가진 균: 폐렴사슬알균, 고초균, 나이세리아균, 파일로리균 등

**수평적 유전자전달 3가지 비교 정리**

| | **접합(conjugation)** | **형질도입(transduction)** | **형질전환(transformation)** |
|---|---|---|---|
| **매개체** | 세균 간 **직접 접촉** (성섬모) | **박테리오파지** (바이러스) | 매개체 없음 (**유리 DNA 단편** 직접 흡수) |
| **전달 DNA** | 플라스미드 DNA (또는 염색체) | 파지에 삽입된 세균 DNA | 환경 중 유리 DNA 단편 |
| **대표 예** | 대장균 F플라스미드, 약제내성 플라스미드 | 독성파지·약독파지에 의한 형질도입 | 그리피스의 폐렴사슬알균 실험 |

### 6. 세균의 병원성

- **병원성 인자(virulence factor)**: 세균은 숙주에 침입한 후 조직에 정착하여 증식함으로써 조직이나 세포를 파괴하거나 독소를 생산하여 **병원성(pathogenicity)**을 발휘하는 인자
- 3가지 병원성 인자: **정착인자, 침입인자, 독성인자**

**세균의 대표적인 병원성 인자 (표 4-7)**

| 작용 | 기능 물질 | 세균 |
|---|---|---|
| **부착·정착** | 섬모 | 대장균 등 |
| | 섬모와 점액다당류 | 녹농균 |
| | 섬유상 적혈구응집소 | 백일해균 |
| | 피브로넥틴 결합단백질 | 고름사슬알균 |
| **항식균작용** | 협막 | 백일해균, 폐렴사슬알균, 폐렴막대균, 인플루엔자균, 수막염균 등 |
| | M단백질 | 고름사슬알균 |
| | 단백질 A | 황색포도알균 |
| | 점액성 물질(알긴산) | 녹농균 |
| **효소** | 히알루론산분해효소 — 히알루론산 분해 | 황색포도알균, 고름사슬알균, 웰치균 등 |
| | 섬유소용해소 — 섬유소 분해 | 황색포도알균, 고름사슬알균 등 |
| | 아교질분해효소 — 콜라겐 분해 | 웰치균, 클로스트리듐균 등 |
| | 혈장응고효소 — 피브린 침착 | 황색포도알균 |
| | 점액분해효소 — 점액 분해 | 대장균, 콜레라균 등 |

#### 1) 정착인자(colonization factor)

- 세균이 병원성을 발휘하려면 무엇보다도 먼저 점막 등의 표적세포에 **부착(adhesion)**하여 증식해야 하며, 이 현상을 **정착(colonization)**이라고 함
- 정착을 위해 세균이 가진 물질 = **정착인자**
- **그람음성균**: 섬모에 각각의 점막상피에 특이적인 **부착소(adhesin)**가 정착인자로 기능
- **그람양성균**: **피브로넥틴 결합단백질**이나 **리포테이코산(lipoteichoic acid)**과 세포 표면의 **피브로넥틴(fibronectin)**과의 결합이 부착을 도움

#### 2) 침입인자

- 조직 침입성이 결여된 병원균은 점막 위 정착에 머무름 (예: **콜레라균** — 점막 위 정착만)
- 일부 세균은 **세포 내부로 더 침입** (예: **이질균** — 상피세포내 침입)
- 세포 내 침입은 없지만, 효소나 독소를 이용하여 **조직을 파괴함으로써 상피 아래로 침입**하는 세균도 있음 (예: **티푸스균**)
- 조직 내 침입을 돕는 효소/독소:
  - 용혈사슬알균의 **트렙토키나아제(streptokinase)**, 포도알균의 **스타필로키나아제(staphylokinase)** → 병소에 형성된 **섬유소를 분해**하여 감염 확산
  - 결합조직 성분인 **엘라스틴, 콜라겐, 히알루론산** 등을 가수분해하는 효소도 감염소 확대에 관여하는 보조적 인자

#### 3) 독성인자 ★★★

- 세균의 생체에 대한 공격력으로서 **독소**가 중요
- 독소는 **내독소(endotoxin)**와 **외독소(exotoxin)**로 나뉨

**외독소와 내독소의 비교 (표 4-8)**

| 비교 항목 | **외독소(exotoxin)** | **내독소(endotoxin)** |
|---|---|---|
| **존재부위** | 균체 **밖으로 분비** | 그람음성균의 **바깥막** |
| **독소의 본체** | **단백질** 또는 펩티드 | **지질다당질(LPS)** (활성은 지질 A 부분이 담당) |
| **가열** | **불안정**(불활성화) | **안정** |
| **항원성** | 항체유도 **용이** | 항체유도 **곤란** |
| **변성독소화** | **가능** (백신으로 사용 가능) | **불가능** |
| **독성** | 독소는 각각 **특이적인 작용**을 가짐 | 생리작용은 모든 균의 내독소가 **거의 동일** |

**(1) 내독소(endotoxin)**

- 그람음성균 바깥막 구성 **지질다당질(LPS)**은 **O측 사슬 + 코어 + 지질 A**(내독소 본체) 부분으로 구성
- **발열작용, 백혈구 활성화작용, 혈소판 응집작용, 보체 활성화작용** 등 다양한 생리활성이 있으며 생체의 감염방어기능을 항진시킴
- 큰포식세포나 가지세포 등에 발현하는 **톨유사수용체(Toll-like receptor, TLR)**가 내독소의 주요 수용체 → **LRR(leucine rich repeat)** 영역으로 특이적 분자 인식
- **내독소 쇼크(endotoxin shock)**: 그람음성균의 LPS 유도 **염증성 사이토카인이 과잉 생산**됨으로써 야기되는 쇼크상태

**(2) 외독소(exotoxin)**

- 그람양성이나 그람음성에 관계없이 각종 균이 외독소를 생산
- 외독소는 **단백질**이며 **열에 불안정**(예외적으로 내열성인 것도 있음)
- 외독소는 특유한 **생물활성(독소작용)**을 나타냄 → 표적에 따라 **신경독, 장관독, 용혈독, 심장독, 세포독** 등으로 분류
- 일부 외독소는 불활성화시킴으로써 **변성독소(toxoid)**로 만들어 **백신**으로서 사용

**세균이 생산하는 주요 외독소 (표 4-9)**

| 독소 | 분류 | 생산균 | 작용과 증상 |
|---|---|---|---|
| **디프테리아독소** | 세포독 | 디프테리아균 | 단백질 합성 억제 |
| **녹농균 외독소 A** | 세포독 | 녹농균 | 단백질 합성 억제 |
| **파상풍독소** | 신경독 | 파상풍균 | **강직마비** |
| **보툴리눔독소** | 신경독 | 보툴리누스균 | **이완마비** |
| **베로독소 / 시가독소** | 세포독, 장관독 | 장자출혈성 대장균 등 / 이질균 | 단백질 합성 억제, 출혈성 설사 |
| **포도알균 창자독소** | 장관독, 신경독 | 황색포도알균 | 설사, 구토 |
| **콜레라독소** | 세포독, 장관독 | 콜레라균 | 설사 |
| **장염비브리오균 내열성 독소** | 세포독, 장관독 | 장염비브리오균 | 설사 |
| **장염비브리오균 내열성 용혈독** | 세포독, 장관독, 심장독 | 장염비브리오균 | 설사 |
| **사슬알균용혈소 O** | 용혈독 | 사슬알균 | — |
| **고름사슬알균 발적독소** | — | 고름사슬알균 | 발열, 발적, 쇼크 |
| **독성쇼크증후군독소** | — | 황색포도알균 | 발열, 발적, 쇼크 |

**(3) 초항원(superantigen)**

- 급성의 심각한 전신 증상을 나타내는 **독성쇼크증후군(toxic shock syndrome)**이나 **성홍열**의 원인 독소
  - 포도알균 **독성쇼크증후군독소(TSST-1)**
  - 사슬알균 **발적독소(erythrogenic toxin)**
- 소량으로 **많은 클론의 T세포를 한꺼번에 활성화** → **대량의 사이토카인**을 생성시킴으로써 생체에 유해한 작용(**발열, 발진, 쇼크** 등)을 일으킴

### 7. 세균 감염병의 진단

- 세균에 의한 감염을 증명하기 위해서는 **병변부위, 분변, 소변, 혈액, 뇌척수액, 가래** 등의 검사재료를 파악
- 식중독: 원인 식품으로 추측되는 것에서 세균을 분리하여 **동정**하는 것이 가장 중요
- 보조적 진단: 세균이나 독소 등의 '존재'를 **유전자적으로 확인**, 또는 특정 세균이나 독소 등에 반응하는 **특이항체나 감작 림프구**의 생산 여부를 검토

#### 1) 분리와 동정

**(1) 검체 채취와 운반**

- 질병의 증상이나 경과를 통해 판단하여 **병원체의 종류 추측** → 그 세균의 종류에 맞는 증균 또는 분리배양법 시행
- 상재균이 다수 존재하는 부위는 **혼합(오염) 방지** 필요
- 채취 후 **즉시 검사 진행**이 이상적, 잠시 보관 시 사멸·증식을 방지
  - 상온에서는 비병원균이 더 빨리 증식하는 경우가 많음 → pH가 보정된 **운반배지(transport medium)**에 채취, **저온 보관** 운반 (임균은 **실온**)
  - **무산소성균**인 경우 공기 접촉 시 사멸 → **환원제를 넣은 연한천배지** 등에 채취, 탄산가스를 충전한 용기 안에 보관

**(2) 분리배양**

- 세균의 분리에서는 우선 **현미경검사**가 중요
- 의심되는 세균만 최대한 **선택적으로 증식시키는 배지**를 사용해서 증균 또는 분리배양
- 세균의 성질에 따라 산소성, 무산소성, 미세산소성, 탄산가스(사슬알균이나 임균은 탄산가스배양기 이용)의 각 배양법으로 생성된 집락에서 의심되는 것을 **순수배양**
- 목적에 따라 액체배지나 사면배지를 사용

**(3) 동정**

- **집락 성질**: 크기, 형태, 색, 발육조건(산소성~무산소성)
- **균 형태**: 크기, 형태, 배열, 편모나 아포 및 협막 형성 여부 등
- **생화학적 성질**: 영양요구성, 당·아미노산 분해능력, 효소 생산능력 등
- **독소 생산성**: 동물이나 배양세포에 대한 독성 등
- **화학적 구성**: 세포벽이나 지질의 구성성분 등
- **면역학적 성질**: 균의 항원성 등
- **유전학적 성질**: GC 함률, DNA나 RNA의 상동성, 16S rRNA의 배열 등

#### 2) 생화학적 검사

| 검사 항목 | 내용 |
|---|---|
| **산소 이용성** | 산소성균인지 무산소성균인지 판정 |
| **용혈성** | 혈액한천 평판배지를 사용하여 용혈성(α, β, γ)을 조사. 일반적으로 **양(sheep)의 혈액** 사용 |
| **색소 생산성** | 고형배지를 사용하여 집락 및 배지에 확산된 색소의 색상을 관찰 |
| **운동성** | 반유동 고층배지에 천자하여 관찰하거나 현적표본을 현미경으로 관찰 |
| **아포 형성** | 아포형성배지로 배양하여 아포를 염색 |
| **협막 형성** | 동물에 접종하여 감염시킨 후 협막을 염색 |
| **당 분해성** | 기초배지에 각종 당(0.5%)과 지시약을 넣고 산생성 유무를 지시약의 색조 변화로 판정 |
| **단백질 분해성** | 젤라틴액화능력, 탈탄산반응, 탈아미노반응(PPA, IPA), 인돌생산능력 등 |
| **요소 분해능력** | 요소를 분해하여 암모니아를 생산하는 효소(**우레아제, urease**)의 검사 |
| **독소 생산성** | 동물이나 배양세포에 작용, 시험관 내 반응으로 판정. 특이항체에 의한 **중화반응**도 중요 |
| **과산화수소분해효소검사(catalase test)** | 과산화수소분해효소(catalase) 생산 판정. 포도알균속, 바실루스속, 장내세균과는 **양성**, 사슬알균속·무산소성균은 **음성** |
| **산화효소검사(oxidase test)** | 나이세리아속 — 인도페놀산화효소, 녹농균·비브리오균 — 시토크롬산화효소 C 생산 여부 판정 |

#### 3) 면역학적 검사

- 세균을 **동정 및 분류**하기 위한 검사
- 환자 검체에 존재하는 미생물, 즉 **항원의 존재를 확인**하는 검사
- 환자 생체 내에서 생산된 미생물에 대한 **특이항체나 감작 림프구의 존재 및 그 양(값)**을 측정하는 검사
- 분리된 세균의 동정을 확실히 하기 위해 이미 알고 있는 **특이항체와의 반응성**을 검토

**주요 면역학적 검사법**

| 검사법 | 내용 |
|---|---|
| **응집반응(agglutination reaction)** | 슬라이드응집반응 등. 세균 동정에 자주 사용 |
| **면역크로마토그래피(immunochromatography)** | 신속진단에 자주 사용 |
| **혈청반응** | 이미 알고 있는 항원을 이용, 단계 희석한 환자 혈청과 반응시켜 특이항체 상승 여부 판정 (예: 비달반응) |
| **침강반응** | 가용성 항원에 사용 |
| **효소항체법(ELISA)** | 가용성 항원에 사용, 일차 항체 + 효소 표지 이차 항체 |
| **중화반응** | 활성을 나타내는 독소나 세균, 바이러스에 사용 |
| **보체결합반응** | 보체결합성 반응에 사용. 제1반응계(본반응: 항원+항체) + 제2반응계(감작 적혈구) |

**피내반응을 이용한 검사**

- **독소에 대한 항체 존재 유무**: 미량의 독소를 피험자 피내에 접종 → 독작용에 의한 발적·종창이 나타나는지 여부로 조사 (예: **딕반응, 시크반응**)
- **지연형 알레르기 존재 유무**: 이미 알고 있는 항원을 피내 접종 → 1~2일 후 발적·종창·경결 출현 여부로 판정 (**피내반응**)
  - 대표적 예: **투베르쿨린반응**
- 그 밖에 세포면역 성립 검사: PHA나 ConA를 사용한 **비특이적 유약화반응**, 특이항원을 사용한 **특이적 유약화반응**, **큰포식세포 이동억제검사** 등

#### 4) 분자생물학적 검사

- 균이나 독소 등 **특정 유전자 존재**를 **PCR법(polymerase chain reaction method)**으로 검출
- 균에서 DNA 추출 및 특정 DNA **제한효소**로 절단 → 그 절단패턴에 따라 동일한 균인지 아닌지를 **펄스장겔전기영동법(pulsed-field gel electrophoresis, PFGE)** 등으로 측정

**PCR법을 이용한 분자생물학적 실험방법 (순서)**
1. 배양 → 단일 집락 선택
2. 혼합 (20μL 증류수)
3. 99℃ 3분간 가열
4. 원심분리
5. 상등액에서 분리한 세균 DNA 채취
6. PCR 방법으로 **16S rRNA** 유전자 증폭 (프라이머, dNTPs, Taq DNA 중합효소, PCR buffer, DNA template)
7. 염기 서열 분석 및 세균 동정

> **PFGE**: 각 레인에 절단된 개별 균 유래의 DNA가 영동되어 바코드처럼 배열됨. 패턴이 동일한 것은 같은 계통일 가능성이 높음

### 8. 항균제 ★★★

#### 1) 화학요법의 정의와 역사

| 용어 | 정의 |
|---|---|
| **화학요법(chemotherapy)** | 화학물질로 병원성 미생물 감염병이나 암 등의 질병을 치유하는 것 |
| **화학요법제(chemotherapeutic agent)** | 화학요법에 사용되는 물질 |
| **항미생물제(antimicrobial agent)** | 화학요법제 중 항미생물활성을 가진 물질 |
| **항생제(antibiotic)** | 다른 미생물에서 유래하는 항미생물질 (예: 푸른곰팡이 → 페니실린) |

- 최근에는 항생제에 대해 다양한 화학적 변형이 이루어져 **반합성 항균제**로서 사용됨

#### 2) 항균제의 특성

**(1) 선택독성(selective toxicity)** ★★

- 숙주인 사람에게는 영향이 없고 **대상 세균에 대해서만 효과를 발휘**할 수 있는 항균제의 성질
- 선택독성을 높이려면 세균과 사람의 세포에서 **차이점을 표적**으로 할 필요가 있음
  - 예) 세균에만 있는 세포벽의 **펩티도글리칸**, 세균과 사람의 세포에서 구조가 다른 **리보솜**
- 항균제 타겟: **자기(진핵세포)에는 없고 비자기(원핵세포)에만 존재하는 것**
  - ① 세균의 **세포벽**에 작용하는 약물
  - ② 세균의 **세포막**에 작용하는 약물
  - ③ 세균의 **핵산합성**에 작용하는 약물
  - ④ 세균의 **단백질합성**에 작용하는 약물

**(2) 정균작용과 살균작용**

| 작용 | 영문 | 설명 | 대표 항균제 |
|---|---|---|---|
| **정균작용** | bacteriostasis | 약물 투여 동안만 세균의 증식을 **억제**, 중단 시 다시 증식 | 설파제, 클로람페니콜 계열, 테트라사이클린 계열 |
| **살균작용** | bactericidal action | 약물 투여 시 세균이 **사멸**, 세포벽·세포막에 장애 유발 | 페니실린, 스트렙토마이신 |

> 살균작용 항균제도 낮은 농도에서는 정균작용을 나타낼 수 있음

**(3) 항균스펙트럼(antimicrobial spectrum)**

- 각 항균제가 효과를 나타내는 **세균의 범위**
- **협역 스펙트럼**: 특정 세균에만 항균 효과
- **광역 스펙트럼**: 여러 세균에 항균 효과
- 항균스펙트럼은 감염병 치료에 이용하는 **약물선택의 지표**로서 중요
- **광역 스펙트럼 항균제 계속 투여 시** → 원인균뿐 아니라 **정상균무리**까지 영향 → 내성균 증식 → **부작용 발생**

#### 3) 항균제의 종류 (작용 타겟별) ★★★

**(1) 세포벽에 작용하는 약물**

- **펩티도글리칸(peptidoglycan)**의 생합성과정을 억제 → 세균에 대해 **높은 선택독성**

**① β-락탐 계열 항균제**
- **벤질페니실린(benzylpenicillin)**으로 대표
- **β-락탐고리** 구조를 가진 항균제
- **페니실린 결합단백질(PBP, penicillin-binding protein)** — 사람에게 부존재 — 에 결합하여 펩티도글리칸의 생합성을 억제

**β-락탐 계열 항균제의 분류 (표 4-10)**

| 분류 | β-락탐아제에 의한 분해성 | 항균스펙트럼 | 주요 항균제 |
|---|---|---|---|
| **페니실린 계열 — 협역** | 페니실린분해효소에 의해 분해 | 그람양성균에 강한 항균활성, 대부분의 그람음성 막대균에는 무효 | 벤질페니실린 |
| **페니실린 계열 — 광역** | 페니실린분해효소에 의해 분해 | 그람양성균에 강한 항균활성, 일부 그람음성균에도 스펙트럼 확대, 피페라실린은 항녹농균활성도 나타냄 | 암피실린, 아목시실린, 피페라실린 |
| **세팔로스포린 계열 — 제1세대** | 페니실린분해효소 내성, 세팔로스포린분해효소에 의해 분해 | 그람양성균에 강한 항균활성, 일부 그람음성균에도 항균활성 | 세팔로틴, 세팔렉신, 세파클러 |
| **세팔로스포린 계열 — 제2세대** | 세팔로스포린분해효소 내성 | 제1세대보다 그람양성균 스펙트럼 축소, 그람음성균에는 스펙트럼 확대 | 세프메타졸, 세포티암 |
| **세팔로스포린 계열 — 제3세대** | 세팔로스포린분해효소 내성 (제2세대보다 더 내성화) | 사슬알균 항균력 강화, 포도알균 항균력 저하, 그람음성균 스펙트럼 더 확대, 일부는 녹농균에도 유효 | 라타목세프, 세포탁심, 세프카펜피복실 |
| **세팔로스포린 계열 — 제4세대** | 세팔로스포린분해효소 내성 | 제3세대보다 포도알균 항균력 강화, 항녹농균 활성 증대 | 세프피롬, 세포조프란, 세페핌 |
| **모노박탐 계열** | β-락탐아제에 잘 분해되지 않지만, 일부 기질확장형 β-락탐아제는 분해 활성을 나타냄 | 그람양성균에는 항균활성 없지만, 녹농균을 포함한 **광범위 그람음성균**에 강한 항균활성 | 아즈트레오남 |
| **카르바페넴 계열** | 페니실린분해효소 내성, 세팔로스포린분해효소 내성, 카르바페넴분해효소에 의해 분해 | 그람양성균에서 음성균까지 **다양한 세균**에 항균활성 | 이미페넴, 파니페넴, 도리페넴, 메로페넴, 비아페넴 |

> 세팔로스포린 계열 약물이 그람음성균에 대한 스펙트럼을 확대하여 **β-락탐아제에 잘 분해되지 않는 방향**으로 개발되었음을 알 수 있다

**② 글리코펩티드 계열 항균제**
- **반코마이신(vancomycin)**, 테이코플라닌(teicoplanin) 등
- 펩티도글리칸 합성과정의 **중간체에 비공유결합** → 펩티도글리칸 합성의 최종단계인 **중합반응, 연결반응을 억제**
- 반코마이신은 **메티실린 내성 황색포도알균(MRSA)** 감염병 치료에 **우선적으로 사용**되는 약물 → 항균제의 적정 사용 관점에서 **혈중농도를 감시**할 필요

**(2) 세포막에 작용하는 약물**

- 세포막에 작용하여 **구멍을 형성** → 막투과성을 변화 → 세포 안의 각종 이온, 아미노산, 단백질 등을 **세포 밖으로 누출**
- **숙주세포의 세포막에도 작용**하기 때문에 **선택독성이 낮아** 부작용에 유의해야 함
- 대표 약물: 암포테리신 B(amphotericin B), 폴리믹신 B(polymyxin B), 콜리스틴(colistin), 니스타틴(nystatin)

**(3) 핵산합성에 작용하는 약물**

| 분류 | 작용기전 |
|---|---|
| **퀴놀론 계열 항균제** | 세균 DNA **선회효소(gyrase)**나 **국소이성화효소 IV(topoisomerase IV)** 활성을 억제 → DNA 복제 억제. 그람음성균에 강한 항균활성 |
| **리팜피신(rifampicin)** | 세균의 **DNA 의존성 RNA 중합효소**에 작용하여 DNA에서 RNA로 **전사되는 것을 억제** |
| **엽산합성경로억제제** | **설파제(sulfa drug)**와 **트리메토프림(trimethoprim)**은 핵산 생합성 보조효소인 **테트라히드로엽산**의 합성과정을 억제. **ST합제** = 설파메톡사졸(sulfamethoxazole)과 트리메토프림을 5:1로 혼합한 약물 |

> 엽산합성경로: PABA → (설파제에 의해 차단) → 디히드로엽산 → (트리메토프림에 의해 차단) → 테트라히드로엽산 → 핵산 합성

**(4) 단백질합성에 작용하는 약물** ★★

- 세균의 **리보솜(70S)**에 선택적으로 작용하여 항균활성을 나타냄

| 계열 | 작용 부위 | 작용기전 | 대표 약물 |
|---|---|---|---|
| **아미노글리코시드 계열** | **30S** 리보솜 | 리보솜 복합체 형성 억제 또는 mRNA 위의 **코돈 오번역** 유발 | 카나마이신, 겐타마이신, 스트렙토마이신 |
| **마크롤라이드 계열** | **50S** 리보솜 (23S rRNA) | **펩티딜 전이효소반응** 억제 | 에리트로마이신, 클라리트로마이신, 아지트로마이신 |
| **테트라사이클린 계열** | **30S** 리보솜 (16S rRNA) | 아미노아실 tRNA가 리보솜 **A위 결합을 억제** | 테트라사이클린, 미노사이클린 |
| **옥사졸리디논 계열** | **50S** 리보솜 | **70S 단백질합성 개시복합체** 형성 억제. MRSA 감염병 치료제 | 리네졸리드 |
| **기타** | **50S** 리보솜 | 린코마이신·클린다마이신: 펩티딜 전이효소반응 억제. 클로람페니콜: 23S rRNA에 결합, 펩티드결합 형성 억제. 퀴누프리스틴/달포프리스틴 합제(3:7) | 린코마이신, 클린다마이신, 클로람페니콜 |

**(5) 항결핵제(antituberculosis drug)**

- 결핵균의 특수한 **세포벽 합성경로**에 관여하는 효소를 표적으로 하는 약물 + 일반 세균에도 효과를 나타내는 항균제로 구성
- **1차 항결핵제(first line drug)**: 항결핵 효과가 좋고 부작용이 적어 **우선적으로 사용**
- **2차 항결핵제(second line drug)**: 항균력은 떨어지나 다제병용에 의한 효과를 기대, 1차 항결핵제를 내성이나 부작용으로 사용할 수 없는 경우 사용

**항결핵제의 작용기전과 부작용 (표 4-11)**

| 구분 | 항결핵제 | 작용기전 | 주요 부작용 | 비고 |
|---|---|---|---|---|
| **1차** | **이소니아지드(isoniazid)** | 지방산 합성효소(FAS II)에 작용하여 세포벽 성분인 **미콜산의 합성을 억제** | 간장애 | 활동기 결핵균에 살균적 작용, **카탈라아제(KatG)**에 의해 활성화되는 전구약물 |
| **1차** | **피라지나미드(pyrazinamide)** | 결핵균 세포 내의 피라진아미드분해효소에 의해 피라진산으로 변환 → **지방산 합성효소(FAS I)**에 작용하여 지방산 합성 억제 | 간장애 | 전구약물 |
| **1차** | **리팜피신(rifampicin)** | **RNA 중합효소를 억제** | 간장애 | CYP3A4 유도능력이 있어 **HIV 감염병 치료제와는 병용 금기** |
| **1차** | **리파부틴(rifabutin)** | RNA 중합효소를 억제 | 간장애 | 리팜피신보다 CYP 유도능력은 낮고 HIV 감염병 치료제와도 **병용 가능** |
| **1차** | **스트렙토마이신(streptomycin)** | **단백질합성을 억제** | 제8뇌신경장애, 콩팥장애 | 부작용인 **난청**으로 인해 최근에는 사용이 감소 |
| **1차** | **에탐부톨(ethambutol)** | 세포벽 성분인 **아라비노갈락탄의 생합성을 억제** | **시신경염, 시력저하** | |
| **2차** | 카나마이신(kanamycin) | 단백질합성을 억제 | 제8뇌신경장애, 콩팥장애 | |
| **2차** | 에티오나미드(ethionamide) | 지방산합성효소(FAS II)에 작용하여 미콜산의 합성을 억제 | 간장애 | 모노옥시다아제(EthA)에 의해 활성화되는 전구약물 |
| **2차** | 엔비오마이신(enviomycin) | 단백질합성 억제작용을 하는 것으로 추정 | 제8뇌신경장애, 콩팥장애 | |
| **2차** | 파라아미노살리실산(PAS) | 엽산 대사의 억제 가능성이 지적 | 무과립구증, 용혈빈혈, 간염, 저인산혈증 | 정균적으로 작용 |
| **2차** | 사이클로세린(cycloserine) | **펩티도글리칸 전구체의 합성을 억제** | 정신착란, 전간유사발작 | |
| **2차** | 레보플록사신(levofloxacin) | **DNA 합성을 억제** | 소화기증상, 경련, 광선과민증 | |

#### 4) 항균제 내성 ★★★

**(1) 정의**

- 같은 균종의 세균이라도 유전자에 다양성이 존재 → 성질의 차이로 표현됨
- 많은 동종의 세균에 대해 **최소억제농도(MIC)**를 측정하면 어떤 농도를 기준으로 정규분포를 나타냄 → **내성균**: 고농도로 벗어난 수치를 보이는 균

**현재 문제가 되는 약제내성균 (표 4-12)**

| 명칭 | 약칭 | 성질 |
|---|---|---|
| **메티실린 내성 황색포도알균** | **MRSA** | PBP의 변이로 β-락탐 계열 항균제 전반에 내성. **mecA 유전자**가 PBP2a를 부호화 |
| **반코마이신 내성 황색포도알균** | **VRSA** | 장알균 등에서 **vanA** 등의 내성유전자를 획득하여 내성화. 세포벽을 두껍게 하여 중간 정도 내성을 획득한 **VISA**도 보고 |
| **반코마이신 내성 장알균** | **VRE** | 원래 β-락탐 계열 약물 대부분에 자연내성, 세팔로스포린 계열 무효. 특히 *Enterococcus faecium*은 암피실린 내성주가 많음 |
| **다제 내성 녹농균** | MDRP | 광역 β-락탐 계열, 플루오로퀴놀론 계열, 아미노글리코시드 계열 **모두에 내성** |
| **다제 내성 임균** | — | 페니실린이나 테트라사이클린 내성에 더해 플루오로퀴놀론 내성화율이 약 80%. 제3세대 세팔로스포린 계열 약물 등에도 내성 보고 증가 |
| **다제 내성 살모넬라** | — | 암피실린, 클로람페니콜, 설파제, 테트라사이클린, 스트렙토마이신 외 세팔로스포린 계열 약물에도 내성 |
| **마크롤라이드 내성 폐렴미코플라스마** | — | 세포벽이 없어 마크롤라이드 계열이 주로 사용되는데, 표적인 **리보솜 RNA에 변이**가 생겨 내성 증가 경향 |
| **다제 내성 세라티아** | — | 다양한 항균제에 자연내성. 제3세대 세팔로스포린, 카르바페넴, 아미노글리코시드, 플루오로퀴놀론 등에도 내성 |
| **β-락탐아제 비생산 암피실린 내성 인플루엔자균** | **BLNAR** | β-락탐아제를 생산하지 않고 β-락탐내성을 획득. **PBP의 변이**에 따른 것 |
| **페니실린 내성 폐렴알균** | PRSP | PBP에 변이가 생기거나 페니실린 결합능력이 낮은 PBP를 획득하여 내성 |
| **약물내성 아시네토박터** | — | 다양한 항균제에 자연내성. 광역 β-락탐 계열, 아미노글리코시드, 플루오로퀴놀론 **3계통 모두에 내성** |

**(2) 내성기전** ★★

세균이 항균제에 내성을 획득하는 기전 5종류:

| 내성기전 | 설명 | 대표 예 |
|---|---|---|
| **① 항균제 불활성화** | 항균제를 **분해 또는 화학적 변형**하여 불활성화 | **β-락탐아제(β-lactamase)**가 β-락탐고리를 가수분해하여 파열 |
| **② 표적 변화** | 세균이 가진 항균제의 **작용부위가 변이**하면 항균제가 작용할 수 없음 | PBP 변이 (MRSA, BLNAR, PRSP) |
| **③ 대체효소 생산** | 항균제가 억제하는 효소 대신 **대체효소를 생산**하여 기능 유지 | **mecA 유전자** → **PBP2a(PBP2')** 생산 (MRSA) |
| **④ 투과성 저하** | 그람음성균 바깥막의 **구멍단백질의 수 감소 또는 결손** → 항균제가 세포질 속으로 들어갈 수 없음 | 카르바페넴 계열 약물에 대한 내성 |
| **⑤ 항균제 배출** | 세포막에 존재하는 **약제배출펌프**가 세포 내 항균제를 세포 밖으로 **능동적으로 배출** → 세포 내 농도 저하 | 다양한 항균제에 대한 내성 |

#### 5) 항균제 부작용

- 항균제가 아무리 선택독성이 강하다 해도 체내 농도가 일정 이상 도달하면 인체에 **부작용**을 일으킬 수 있음

**대표적인 부작용 3가지**

| 부작용 | 설명 |
|---|---|
| **장기장애** | 아미노글리코시드 계열(스트렙토마이신)에 의한 **청각신경장애**, 클로람페니콜에 의한 **재생불량빈혈·과립백혈구감소증**, 에리트로마이신·리팜피신에 의한 **간장애** 등 |
| **균교대현상(microbial substitution)** | 광역 스펙트럼 항균제 사용 시 **정상균무리의 균형이 파괴** → 해당 항균제에 내성을 가지는 세균이 비정상적으로 증식하는 현상 |
| **약물알레르기(drug allergy)** | 페니실린 계열, 세팔로스포린 계열에 의한 **아나필락시스(anaphylaxis)** 등 |

**주요 항균제의 부작용 (표 4-13)**

| 항균제 | 주요 부작용 |
|---|---|
| 페니실린 계열, 세팔로스포린 계열 | **아나필락시스** 유사반응(쇼크) |
| 마크롤라이드 계열, 테트라사이클린 계열 | **간기능장애** |
| 아미노글리코시드 계열, 폴리펩티드 계열, 세팔로스포린 계열 | **콩팥기능장애** |
| 클로람페니콜 | **혈액장애** (재생불량빈혈 등) |
| 아미노배당체 계열 | **제8뇌신경장애** (난청, 귀울림, 감각이상) |
| 폴리펩티드 계열 | **신경·근육차단** (호흡억제) |
| 세팔로스포린 계열 일부 (라타목세프, 세프부페라존) | **디설피람 작용** (술과 함께 복용하지 않도록 주의) |
| 린코마이신, 암피실린, 제3세대 세팔로스포린 계열 | **거짓막창자염** — 창자 내에 비정상적으로 증식한 *Clostridium difficile*가 생산하는 독소의 영향 |
| 광역 스펙트럼 항생물질 (테트라사이클린, 암피실린, 세팔로스포린 계열) | 장내세균 억제에 의한 **비타민 K 결핍** → 간에서 혈액응고인자의 생산이 저하되어 **출혈경향** |
